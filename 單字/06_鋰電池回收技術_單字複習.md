<a id="環境與回收"></a>
## 主題一：環境與回收

- 1. **ขยะ** [khà-yà] 廢棄物/垃圾 (waste)
  - 字構：諧音「卡亞」→ 卡亞卡亞的垃圾推成山
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ขยะแบตเตอรี่</span>（電池廢棄物）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปัญหาขยะ</span>（廢棄物問題）
  - 例：ปัญหา  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ขยะ</span>  แบตเตอรี่  กำลัง  เพิ่มขึ้น  ทั่วโลก（電池廢棄物問題正在全球增加）

- 2. **รีไซเคิล** [rii-sai-khoen] 回收再利用 (recycle)
  - 記憶法：直接音譯自英文 recycle
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">รีไซเคิล แบตเตอรี่</span>（回收電池）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">โรงงาน รีไซเคิล</span>（回收工廠）
  - 例：ญี่ปุ่น  มี  โรงงาน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">รีไซเคิล</span>  แบตเตอรี่  ที่  ทันสมัย（日本有先進的電池回收工廠）

- 3. **กู้คืน** [kûu-khuuen] 回收/取回 (recover/retrieve)
  - 字構：กู้（救回）+ คืน（還回）= 把資源救回來還給自己
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กู้คืน ลิเทียม</span>（回收鋰）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">อัตราการกู้คืน</span>（回收率）
  - 例：บริษัท  สามารถ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กู้คืน</span>  ลิเทียม  ได้  ถึง  90%（公司能夠回收高達90%的鋰）

- 4. **นำกลับมาใช้ใหม่** [nam-klàp-maa-chái-mài] 循環再利用 (reuse/recycle)
  - 字構：นำ（帶）+ กลับมา（回來）+ ใช้ใหม่（重新使用）= 帶回來重新使用
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">นำ โลหะ กลับมาใช้ใหม่</span>（循環再利用金屬）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">นำกลับมาใช้ใหม่ได้ 90%</span>（可循環再利用90%）
  - 例：โลหะ  ใน  แบตเตอรี่  เก่า  สามารถ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">นำ  กลับมา  ใช้  ใหม่</span>  ได้（舊電池中的金屬可以循環再利用）

- 5. **การปล่อยก๊าซ** [kaan-plòi-kàat] 氣體排放 (gas emission)
  - 字構：การ（名詞化）+ ปล่อย（排放）+ ก๊าซ（氣體）= 排放氣體的行為
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ลด การปล่อยก๊าซ</span>（減少氣體排放）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">การปล่อยก๊าซ CO2</span>（CO2排放）
  - 例：กระบวนการ  ใหม่  ช่วย  ลด  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">การปล่อยก๊าซ</span>  CO2  ได้  ประมาณ  40%（新工藝有助於減少約40%的CO2排放）

- 6. **คาร์บอนไดออกไซด์** [khaa-bawn-dai-àwk-sàit] 二氧化碳 (carbon dioxide / CO2)
  - 記憶法：音譯自英文 carbon dioxide，縮寫 CO2
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ลด คาร์บอนไดออกไซด์</span>（減少二氧化碳）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปล่อย CO2</span>（排放CO2）
  - 例：การ  รีไซเคิล  ช่วย  ลด  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">คาร์บอนไดออกไซด์</span>  ที่  ปล่อย  สู่  บรรยากาศ（回收有助於減少排放到大氣的二氧化碳）

- 7. **ปลดระวาง** [plòt-rá-waang] 退役/除役 (decommission/retire)
  - 字構：ปลด（卸除）+ ระวาง（負荷/任務）= 卸除任務=退役
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">รถยนต์ ที่ปลดระวาง</span>（退役車輛）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">แบตเตอรี่ ที่ปลดระวาง</span>（退役電池）
  - 例：รถยนต์  ไฟฟ้า  ที่  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ปลดระวาง</span>  แล้ว  ควร  ถูก  รีไซเคิล（退役的電動車應該被回收）

- 8. **หมดอายุ** [mòt-aa-yú] 到期/報廢 (expired/end of life)
  - 字構：หมด（耗盡）+ อายุ（壽命）= 壽命耗盡=報廢
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">แบตเตอรี่ หมดอายุ</span>（電池報廢）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">หมดอายุ การใช้งาน</span>（使用壽命終止）
  - 例：แบตเตอรี่  ที่  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">หมดอายุ</span>  จำเป็น  ต้อง  นำ  ไป  รีไซเคิล（報廢的電池必須送去回收）

<a id="礦物與材料"></a>
## 主題二：礦物與材料

- 9. **ลิเทียม** [lí-thiiam] 鋰 (lithium)
  - 記憶法：音譯自英文 lithium，元素符號 Li
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">สกัด ลิเทียม</span>（提取鋰）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">แร่ ลิเทียม</span>（鋰礦）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ลิเทียม</span>  เป็น  ส่วนประกอบ  สำคัญ  ของ  แบตเตอรี่  รถ  EV（鋰是電動車電池的重要成分）

- 10. **โลหะ** [loo-hà] 金屬 (metal)
  - 字構：諧音「羅哈」→ 羅哈的礦山富含金屬
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">โลหะมีค่า</span>（貴重金屬）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">โลหะ ที่ไม่ใช่เหล็ก</span>（非鐵金屬）
  - 例：แบตเตอรี่  ประกอบด้วย  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">โลหะ</span>  หลาย  ชนิด  ที่  มีค่า（電池由多種貴重金屬組成）

- 11. **แร่ธาตุ** [râe-thâat] 礦物/礦石 (mineral/ore)
  - 字構：แร่（礦）+ ธาตุ（元素）= 礦物元素
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">นำเข้า แร่ธาตุ</span>（進口礦物）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">แร่ธาตุ ที่จำเป็น</span>（必要的礦物）
  - 例：ญี่ปุ่น  ต้อง  นำเข้า  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">แร่ธาตุ</span>  เกือบ  ทุก  ชนิด  จาก  ต่างประเทศ（日本幾乎所有礦物都必須從國外進口）

- 12. **โคบอลต์** [khoo-bàwt] 鈷 (cobalt)
  - 記憶法：音譯自英文 cobalt，元素符號 Co
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">แร่ โคบอลต์</span>（鈷礦）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">รีไซเคิล โคบอลต์</span>（回收鈷）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">โคบอลต์</span>  และ  นิกเกิล  เป็น  โลหะ  ที่  ใช้  ใน  แบตเตอรี่  EV（鈷和鎳是電動車電池中使用的金屬）

- 13. **นิกเกิล** [ní-koen] 鎳 (nickel)
  - 記憶法：音譯自英文 nickel，元素符號 Ni
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">แร่ นิกเกิล</span>（鎳礦）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">นิกเกิล โคบอลต์ ลิเทียม</span>（鎳鈷鋰三元）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">นิกเกิล</span>  ถูก  นำ  ไป  ใช้  ใน  แบตเตอรี่  รถยนต์  ไฟฟ้า  จำนวน  มาก（鎳被大量用於電動車電池）

- 14. **ลิเทียมไฮดรอกไซด์** [lí-thiiam-hái-drawng-sàit] 氫氧化鋰 (lithium hydroxide / LiOH)
  - 字構：ลิเทียม（鋰）+ ไฮดรอกไซด์（氫氧化物）= 氫氧化鋰
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">สกัด ลิเทียมไฮดรอกไซด์</span>（提取氫氧化鋰）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">LiOH บริสุทธิ์</span>（純氫氧化鋰）
  - 例：กระบวนการ  ใหม่  สกัด  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ลิเทียมไฮดรอกไซด์</span>  ออก  มา  ได้  อย่าง  มีประสิทธิภาพ（新工藝能有效提取氫氧化鋰）

- 15. **ผง** [phǒng] 粉末 (powder)
  - 字構：諧音「砰」→ 敲碎後砰的一聲變成粉末
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">บด เป็นผง</span>（研磨成粉）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ผง โลหะ</span>（金屬粉末）
  - 例：แบตเตอรี่  เก่า  ถูก  บด  ให้  เป็น  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ผง</span>  ก่อน  การ  สกัด（舊電池在提取前被研磨成粉末）

<a id="工業製程"></a>
## 主題三：工業製程

- 16. **การสกัด** [kaan-sà-kàt] 提取/萃取 (extraction)
  - 字構：การ（名詞化）+ สกัด（截取/提取）= 提取的過程
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">การสกัด ลิเทียม</span>（提取鋰）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">วิธีการ สกัด</span>（提取方法）
  - 例：วิธีการ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">สกัด</span>  ใหม่  ทำให้  ได้  ลิเทียม  มากขึ้น  ถึง  90%（新的提取方法獲得高達90%的鋰）

- 17. **กระบวนการ** [krà-buan-kaan] 流程/過程 (process)
  - 字構：กระบวน（行列/序列）+ การ（事）= 有順序的事情過程
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กระบวนการ รีไซเคิล</span>（回收流程）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กระบวนการ ใหม่</span>（新流程）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กระบวนการ</span>  ใหม่  นี้  ช่วย  ลด  การ  ปล่อย  CO2  ได้  มาก（這個新流程大幅減少CO2排放）

- 18. **สารเคมี** [sǎan-khee-mii] 化學物質 (chemical)
  - 字構：สาร（物質）+ เคมี（化學）= 化學物質
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">สารเคมี ใหม่</span>（新化學物質）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">คิดค้น สารเคมี</span>（研發化學物質）
  - 例：ทีมงาน  คิดค้น  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">สารเคมี</span>  ใหม่  ใน  การ  สกัด  ลิเทียม（團隊研發了新的化學物質用於提取鋰）

- 19. **การกลั่น** [kaan-klân] 精煉/蒸餾 (refining/distillation)
  - 字構：การ + กลั่น（精煉過濾）= 精煉的過程
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">การกลั่น แบบดั้งเดิม</span>（傳統精煉法）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">วิธีการ กลั่น</span>（精煉方法）
  - 例：วิธี  ใหม่  แทนที่  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">การกลั่น</span>  แบบ  ดั้งเดิม  ที่  ปล่อย  CO2  มาก（新方法取代了排放大量CO2的傳統精煉法）

- 20. **การบำบัดทางเคมี** [kaan-bam-bàt-thaang-khee-mii] 化學處理 (chemical treatment)
  - 字構：การบำบัด（處理/治療）+ ทางเคมี（以化學方式）= 化學處理
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">การบำบัด ด้วยน้ำ</span>（水法處理）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">การบำบัด ทางเคมี</span>（化學處理法）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">การบำบัด  ทางเคมี</span>  ด้วย  น้ำ  ช่วย  ดึง  ลิเทียม  ออก  มา（水法化學處理幫助提取出鋰）

- 21. **แยกชิ้นส่วน** [yâek-chín-sùan] 拆解/分解 (disassemble)
  - 字構：แยก（分開）+ ชิ้นส่วน（零件/部件）= 把零件分開拆解
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">แยกชิ้นส่วน แบตเตอรี่</span>（拆解電池）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ขั้นตอน การแยกชิ้นส่วน</span>（拆解步驟）
  - 例：ขั้นตอน  แรก  คือ  การ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">แยกชิ้นส่วน</span>  แบตเตอรี่  เก่า（第一步是拆解舊電池）

- 22. **เผา** [phǎo] 焚燒/燃燒 (burn/incinerate)
  - 字構：諧音「漂」→ 燃燒的火焰往上飄
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เผา ชิ้นส่วน</span>（焚燒零件）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เผา เพื่อกำจัด</span>（焚燒以清除）
  - 例：ชิ้นส่วน  ที่  ไม่ใช่  โลหะ  ถูก  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เผา</span>  ออก  ก่อน  การ  สกัด（非金屬零件在提取前先被焚燒清除）

- 23. **กำจัด** [kam-jàt] 清除/消除 (eliminate/dispose of)
  - 字構：กำ（做）+ จัด（整理/除去）= 做到徹底除去
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กำจัด ขยะ</span>（清除廢棄物）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กำจัด ส่วนประกอบ</span>（清除成分）
  - 例：กระบวนการ  เผา  ช่วย  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กำจัด</span>  ส่วนประกอบ  ที่  ไม่  ต้องการ  ออก  ไป（焚燒過程幫助清除不需要的成分）

- 24. **ส่วนประกอบ** [sùan-prà-kàwp] 成分/組成部分 (component)
  - 字構：ส่วน（部分）+ ประกอบ（組成）= 組成的部分
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ส่วนประกอบ หลัก</span>（主要成分）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ส่วนประกอบ ที่ไม่ใช่โลหะ</span>（非金屬成分）
  - 例：แบตเตอรี่  ประกอบด้วย  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ส่วนประกอบ</span>  ที่  ซับซ้อน  หลาย  อย่าง（電池由許多複雜的成分組成）

- 25. **คิดค้น** [khít-khón] 研發/發明 (invent/develop)
  - 字構：คิด（思考）+ ค้น（探索）= 思考探索出新方法
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">คิดค้น วิธีการ</span>（研發方法）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">คิดค้น สารเคมีใหม่</span>（研發新化學物質）
  - 例：ทีมงาน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">คิดค้น</span>  วิธีการ  ใหม่  ใน  การ  สกัด  ลิเทียม（團隊研發了新的鋰提取方法）

- 26. **ความก้าวหน้า** [khwaam-kâao-nâa] 進步/突破 (progress/advancement)
  - 字構：ความ（名詞化）+ ก้าวหน้า（向前步進）= 向前進步的狀態
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ความก้าวหน้า ครั้งสำคัญ</span>（重大突破）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ความก้าวหน้า ทางเทคโนโลยี</span>（技術進步）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ความก้าวหน้า</span>  ครั้งนี้  อาจ  ช่วย  แก้ปัญหา  การขาดแคลน  ลิเทียม（這次突破可能有助於解決鋰短缺問題）

- 27. **เซลล์แบตเตอรี่** [seen-bàet-tòe-rii] 電池芯 (battery cell)
  - 字構：เซลล์（音譯 cell 細胞/芯）+ แบตเตอรี่（電池）= 電池芯
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เซลล์แบตเตอรี่ ที่ใช้แล้ว</span>（用過的電池芯）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">สกัด จากเซลล์แบตเตอรี่</span>（從電池芯提取）
  - 例：ลิเทียม  ถูก  สกัด  ออก  มา  จาก  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เซลล์  แบตเตอรี่</span>  ที่  ใช้  แล้ว（鋰從用過的電池芯中提取出來）

<a id="商業與貿易"></a>
## 主題四：商業與貿易

- 28. **อุปทาน** [ùp-pà-taan] 供應/供給 (supply)
  - 字構：來自梵語，諧音「烏帕探」→ 烏帕探索供應鏈
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">อุปทาน ขาดแคลน</span>（供應短缺）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">อุปทาน ลิเทียม</span>（鋰的供應）
  - 例：เมื่อ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">อุปทาน</span>  ขาดแคลน  ราคา  ก็  จะ  สูงขึ้น（當供應短缺時，價格就會上漲）

- 29. **ขาดแคลน** [khàat-khlaen] 短缺/匱乏 (scarce/shortage)
  - 字構：ขาด（缺少）+ แคลน（稀少）= 缺少稀少=供應短缺
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ขาดแคลน ลิเทียม</span>（鋰短缺）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปัญหา ขาดแคลน</span>（短缺問題）
  - 例：การ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ขาดแคลน</span>  แร่ธาตุ  ส่งผล  ต่อ  อุตสาหกรรม  EV（礦物短缺影響電動車產業）

- 30. **นำเข้า** [nam-khâo] 進口 (import)
  - 字構：นำ（帶）+ เข้า（進入）= 帶進來=進口
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">นำเข้า แร่ธาตุ</span>（進口礦物）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">สินค้า นำเข้า</span>（進口商品）
  - 例：ญี่ปุ่น  ต้อง  พึ่งพา  การ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">นำเข้า</span>  แร่ธาตุ  จาก  ต่างประเทศ（日本必須依賴從國外進口礦物）

- 31. **ส่งออก** [sòng-àwk] 出口 (export)
  - 字構：ส่ง（送）+ ออก（出）= 送出去=出口
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ส่งออก รถยนต์</span>（出口汽車）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">การ ส่งออก</span>（出口行為）
  - 例：รถยนต์  ไฟฟ้า  เก่า  จำนวน  มาก  ถูก  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ส่งออก</span>  ไป  ต่างประเทศ（大量舊電動車被出口到國外）

- 32. **บริษัทในเครือ** [baw-rí-sàt-nai-khruuea] 子公司/旗下企業 (subsidiary/affiliated company)
  - 字構：บริษัท（公司）+ ในเครือ（在集團網絡中）= 集團旗下的公司
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">บริษัทในเครือ ของ</span>（某集團的子公司）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">บริษัทแม่</span>（母公司）
  - 例：JX Metals Circular Solutions  เป็น  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">บริษัทในเครือ</span>  ของ  กลุ่ม  JX Metals（JX Metals Circular Solutions是JX Metals集團的子公司）

- 33. **ผู้ผลิต** [phûu-phalìt] 製造商/生產者 (manufacturer/producer)
  - 字構：ผู้（人/主體）+ ผลิต（生產）= 生產的人或商家
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ผู้ผลิต แบตเตอรี่</span>（電池製造商）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ผู้ผลิต รถยนต์</span>（汽車製造商）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ผู้ผลิต</span>  ต้อง  รับผิดชอบ  ต่อ  การ  รีไซเคิล  แบตเตอรี่（製造商必須對電池回收負責）

- 34. **ผู้นำเข้า** [phûu-nam-khâo] 進口商 (importer)
  - 字構：ผู้（主體）+ นำเข้า（進口）= 進行進口的商家
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ผู้นำเข้า และ ผู้ผลิต</span>（進口商和製造商）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">หน้าที่ ผู้นำเข้า</span>（進口商職責）
  - 例：กฎหมาย  ใหม่  กำหนด  ให้  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ผู้นำเข้า</span>  ต้อง  รวบรวม  แบตเตอรี่  เก่า（新法律規定進口商必須收集舊電池）

- 35. **เป้าหมาย** [pâo-mǎai] 目標 (goal/target)
  - 字構：เป้า（靶/目的地）+ หมาย（標記）= 標記好的目標
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">บรรลุ เป้าหมาย</span>（達成目標）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตั้ง เป้าหมาย</span>（設定目標）
  - 例：รัฐบาล  ตั้ง  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เป้าหมาย</span>  กู้คืน  ลิเทียม  70%  ภายใน  ปี  2030（政府設定2030年前回收70%鋰的目標）

## 主題五：政策法規

- 36. **กฎหมาย** [kòt-mǎai] 法律 (law)
  - 字構：กฎ（規則）+ หมาย（標記/令）= 標記好的規則=法律
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กฎหมาย ใหม่</span>（新法律）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กฎหมาย รีไซเคิล</span>（回收法律）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กฎหมาย</span>  ใหม่  จะ  มี  ผล  บังคับ  ใช้  ใน  ปี  นี้（新法律將於今年生效施行）

- 37. **มีผลบังคับใช้** [mii-phon-bang-khàp-châi] 生效/施行 (take effect/be enforced)
  - 字構：มีผล（有效力）+ บังคับ（強制）+ ใช้（使用）= 強制生效
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กฎหมาย มีผลบังคับใช้</span>（法律生效）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">วันที่ มีผลบังคับใช้</span>（生效日期）
  - 例：กฎหมาย  รีไซเคิล  จะ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">มีผลบังคับใช้</span>  ในปีนี้（電池回收法律將於今年生效）

- 38. **รวบรวม** [rûap-ruam] 收集/彙整 (collect/gather)
  - 字構：รวบ（聚攏）+ รวม（合併）= 聚攏合而為一
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">รวบรวม แบตเตอรี่</span>（收集電池）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ช่องทาง การรวบรวม</span>（收集管道）
  - 例：ผู้ผลิต  ต้อง  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">รวบรวม</span>  แบตเตอรี่  เก่า  จาก  ผู้บริโภค（製造商必須向消費者收集舊電池）

- 39. **กำหนดให้** [kam-nòt-hâi] 規定/要求（某人做某事）(stipulate/require)
  - 字構：กำหนด（規定）+ ให้（讓/要求）= 規定讓某人做
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กฎหมาย กำหนดให้</span>（法律規定）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กำหนดให้ ผู้ผลิต</span>（要求製造商）
  - 例：กฎหมาย  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กำหนดให้</span>  ผู้ผลิต  ต้อง  รับ  แบตเตอรี่  เก่า  คืน（法律規定製造商必須回收舊電池）

- 40. **กำหนดการ** [kam-nòt-kaan] 時間表/計畫 (schedule/timeline)
  - 字構：กำหนด（設定）+ การ（計畫/事）= 設定好的計畫時間
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เร็วกว่า กำหนดการ</span>（早於時間表）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตาม กำหนดการ</span>（按照時間表）
  - 例：ตัวเลข  90%  ถือ  ว่า  เร็ว  กว่า  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กำหนดการ</span>  มาก（90%這個數字被認為比時間表提早了很多）

- 41. **บรรลุ** [ban-lu] 達成/實現 (achieve/accomplish)
  - 字構：來自梵語，諧音「班路」→ 走完這條路就達成了目標
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">บรรลุ เป้าหมาย</span>（達成目標）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">บรรลุ ผล</span>（達成成果）
  - 例：ทีมงาน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">บรรลุ</span>  เป้าหมาย  การ  กู้คืน  ลิเทียม  90%（團隊達成了90%鋰回收的目標）

<a id="技術動詞"></a>
## 主題六：技術動詞

- 42. **พึ่งพาตนเอง** [phûeng-phaa-ton-eeng] 自給自足/自力更生 (self-reliance)
  - 字構：พึ่งพา（依賴）+ ตนเอง（自己）= 依賴自己=自給自足
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">พึ่งพาตนเอง ให้มากขึ้น</span>（更加自給自足）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">การ พึ่งพาตนเอง</span>（自力更生）
  - 例：ญี่ปุ่น  พยายาม  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">พึ่งพาตนเอง</span>  ด้าน  แร่ธาตุ  ให้  มากขึ้น（日本努力在礦物方面更加自給自足）

- 43. **เอนเอียง** [een-iang] 傾向/偏向 (lean towards/incline)
  - 字構：เอน（傾斜）+ เอียง（偏斜）= 傾斜偏向某方
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เอนเอียง ไปทาง</span>（傾向某方）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เอนเอียง มากกว่า</span>（更傾向於）
  - 例：ญี่ปุ่น  กำลัง  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เอนเอียง</span>  ไป  ทาง  การ  รีไซเคิล  มากขึ้น（日本越來越傾向於走回收這條路）

- 44. **แทนที่** [thaaen-thîi] 取代/替代 (replace/substitute)
  - 字構：แทน（代替）+ ที่（位置）= 取代那個位置
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">แทนที่ สารเคมีเดิม</span>（取代舊化學物質）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เข้า แทนที่</span>（進入取代）
  - 例：สารเคมี  ใหม่  เข้า  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">แทนที่</span>  สารเคมี  ที่  ใช้  ใน  การ  กลั่น  แบบ  เดิม（新化學物質取代了傳統精煉中使用的化學物質）

- 45. **ช่องทาง** [châwng-thaang] 管道/渠道 (channel/route)
  - 字構：ช่อง（孔洞/通道）+ ทาง（路）= 可通行的渠道
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ช่องทาง การรวบรวม</span>（收集管道）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ช่องทาง อย่างเป็นทางการ</span>（正式渠道）
  - 例：มี  เพียง  14%  ที่  ผ่าน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ช่องทาง</span>  รวบรวม  อย่าง  เป็น  ทางการ（只有14%通過正式收集管道）

- 46. **เทียบกับ** [thîap-kàp] 與…相比/相較於 (compare to/compared with)
  - 字構：เทียบ（比較）+ กับ（與）= 與某物作比較
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เทียบกับ วิธีเก่า</span>（與舊方法相比）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เมื่อ เทียบกับ</span>（當與…相比）
  - 例：วิธี  ใหม่  ลด  CO2  ได้  40%  เมื่อ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เทียบกับ</span>  วิธีการ  แบบ  เก่า（與舊方法相比，新方法減少40%的CO2）

- 47. **อย่างเป็นทางการ** [yâang-pen-thaang-kaan] 正式地/官方地 (formally/officially)
  - 字構：อย่าง（方式）+ เป็นทางการ（正式的）= 以正式方式進行
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ช่องทาง อย่างเป็นทางการ</span>（正式管道）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">การรวบรวม อย่างเป็นทางการ</span>（正式收集）
  - 例：การ  รวบรวม  แบตเตอรี่  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">อย่าง  เป็น  ทางการ</span>  ยัง  มี  สัดส่วน  น้อย（正式的電池收集比例仍然很低）

- 48. **ดำเนินการ** [dam-noen-kaan] 進行/執行 (operate/carry out)
  - 字構：ดำเนิน（推展/進行）+ การ（事）= 推展進行某事
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ดำเนินการ ผ่านประเทศจีน</span>（透過中國進行）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ดำเนินการ กู้คืน</span>（進行回收）
  - 例：ใน  อดีต  การ  กลั่น  แร่ธาตุ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ดำเนินการ</span>  ผ่าน  ประเทศจีน  เป็น  หลัก（過去礦物精煉主要透過中國進行）

- 49. **ขนส่ง** [khǒn-sòng] 運輸/運送 (transport)
  - 字構：ขน（搬運）+ ส่ง（送達）= 搬運送達
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">การ ขนส่ง แบตเตอรี่</span>（電池運輸）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปัญหา การขนส่ง</span>（運輸問題）
  - 例：ปัญหา  ใหญ่  คือ  การ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ขนส่ง</span>  แบตเตอรี่  ไป  ยัง  โรงงาน  รีไซเคิล（最大的問題是將電池運輸到回收工廠）

- 50. **เปิดเผย** [pòet-phǒei] 公開/揭露 (reveal/disclose)
  - 字構：เปิด（開）+ เผย（揭露）= 打開揭露出來
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เปิดเผย กระบวนการ</span>（公開流程）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">สื่อ เปิดเผย</span>（媒體揭露）
  - 例：สื่อ  ญี่ปุ่น  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เปิดเผย</span>  กระบวนการ  จริง  ของ  โรงงาน  ออก  มา（日本媒體公開了工廠的實際流程）
