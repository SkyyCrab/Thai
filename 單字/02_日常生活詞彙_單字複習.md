<a id="量詞"></a>
## 主題一：量詞

- 1. **แผ่น** [phɛ̀ɛn] 張/片（平薄物量詞）(flat piece/sheet)
  - 字構：想像薄薄一片攤開 → 紙、餅乾、CD 都用這個量詞
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กระดาษ แผ่น</span>（一張紙）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ขนมปัง แผ่น</span>（一片吐司）
  - 例：ช่วย  ฉีก  กระดาษ  ให้  สัก  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">แผ่น</span>  ได้  ไหม（可以幫我撕一張紙嗎）

- 2. **ผืน** [phʉ̌ʉn] 一塊/件（布、田、席量詞）(piece for cloth/flat land)
  - 字構：ผ/p/ + 展開鋪平的意象 → 布匹、田地等大型平鋪物件的量詞
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ผ้า ผืน</span>（一塊布）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ทุ่งนา ผืน</span>（一片田）
  - 例：เธอ  ซื้อ  ผ้า  มา  สอง  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ผืน</span>  เพื่อ  ตัด  ชุด（她買了兩塊布來做衣服）

- 3. **ฟอง** [fɔɔng] 顆（蛋量詞）/ 泡泡 (classifier for eggs; bubble)
  - 字構：ฟอง = 泡泡，圓圓的 → 蛋也圓，共用量詞；ฟองสบู่ = 肥皂泡
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ไข่ ฟอง</span>（一顆蛋）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ฟอง สบู่</span>（肥皂泡）
  - 例：สูตร  นี้  ต้องการ  ไข่  สาม  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ฟอง</span>（這個食譜需要三顆蛋）

<a id="文書辦公"></a>
## 主題二：文書辦公

- 4. **กระดาษ** [krà-dàat] 紙 (paper)
  - 字構：諧音「嘎打」→ 打字機在紙上嘎嗒作響的聲音
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กระดาษ A4</span>（A4紙）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กระดาษ ทด</span>（草稿紙）
  - 例：ลืม  เอา  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กระดาษ</span>  มา  เลย  ต้อง  ยืม  เพื่อน（忘了帶紙，只好借朋友的）

- 5. **กรอก** [krɔ̀ɔk] 填寫/填入 (fill in)
  - 字構：เสียง "กรอก" คล้าย เทน้ำ → 把資料「倒進」表格欄位
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กรอก แบบฟอร์ม</span>（填寫表格）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กรอก ข้อมูล</span>（填入資料）
  - 例：โปรด  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กรอก</span>  ข้อมูล  ส่วนตัว  ให้  ครบถ้วน（請完整填寫個人資料）

- 6. **ให้ครบถ้วน** [hâi-khróp-thûan] 完整地/齊全地 (completely/in full)
  - 字構：ให้（使）+ ครบ（齊全）+ ถ้วน（完整）= 讓每個欄位都填滿
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กรอก ให้ครบถ้วน</span>（完整填寫）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตอบ ให้ครบถ้วน</span>（完整作答）
  - 例：กรุณา  กรอก  แบบฟอร์ม  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ให้ครบถ้วน</span>  ก่อน  ส่ง（提交前請完整填寫表格）

<a id="農業永續"></a>
## 主題三：農業永續

- 7. **ทุ่งนา** [thûng-naa] 稻田/田野 (rice field/paddy field)
  - 字構：ทุ่ง（廣闊原野）+ นา（稻田）= 一望無際的稻田
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ทุ่งนา เขียวขจี</span>（翠綠稻田）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กลาง ทุ่งนา</span>（田野中央）
  - 例：ยืน  มอง  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ทุ่งนา</span>  เขียวขจี  กว้าง  ไกล  สุดตา（眺望遠處一望無際的翠綠稻田）

- 8. **เกษตร** [kà-sèet] 農業/農事 (agriculture)
  - 字構：มาจากสันสกฤต kṣetra（土地/田地）→ 耕作土地的事業
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เกษตร อินทรีย์</span>（有機農業）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เกษตรกร</span>（農民）
  - 例：ประเทศไทย  ใช้  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เกษตร</span>  เป็น  รากฐาน  เศรษฐกิจ  มา  ช้านาน（泰國長期以農業為經濟基礎）

- 9. **อินทรีย์** [in-sii] 有機 (organic)
  - 字構：มาจากสันสกฤต indriya（感官/本質）→ 現代語義轉為「有機、天然、無化學」
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เกษตร อินทรีย์</span>（有機農業）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ผัก อินทรีย์</span>（有機蔬菜）
  - 例：ผัก  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">อินทรีย์</span>  ปลูก  โดย  ไม่  ใช้  สารเคมี  ใด  ๆ（有機蔬菜不使用任何化學物質種植）

- 10. **อย่างยั่งยืน** [yàang-yâng-yuun] 永續地/持久地 (sustainably)
  - 字構：อย่าง（方式）+ ยั่งยืน（持久穩固；ยืน = 站穩）= 以持久方式進行
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">พัฒนา อย่างยั่งยืน</span>（永續發展）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ทำ เกษตร อย่างยั่งยืน</span>（永續農耕）
  - 例：เป้าหมาย  คือ  การ  ทำ  เกษตร  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">อย่างยั่งยืน</span>  เพื่อ  ลด  การ  ใช้  สารเคมี（目標是以永續農耕方式減少化學物質的使用）

<a id="圖書知識"></a>
## 主題四：圖書知識

- 11. **ห้องสมุด** [hɔ̂ng-sà-mùt] 圖書館 (library)
  - 字構：ห้อง（房間）+ สมุด（書冊/本子）= 存放書冊的房間
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ไป ห้องสมุด</span>（去圖書館）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ยืม หนังสือ จาก ห้องสมุด</span>（從圖書館借書）
  - 例：ฉัน  มักจะ  ไป  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ห้องสมุด</span>  เพื่อ  ค้นคว้า  ข้อมูล（我常去圖書館查閱資料）

- 12. **ยืม** [yʉ̀ʉm] 借 (borrow)
  - 字構：เสียงสั้น นุ่ม → 輕輕一借，短促而無負擔；借了要記得還
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ยืม หนังสือ</span>（借書）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ยืม เงิน</span>（借錢）
  - 例：ขอ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ยืม</span>  พจนานุกรม  สักเล่ม  ได้  ไหม（可以借一本字典嗎）

- 13. **พจนานุกรม** [phót-jà-naa-nú-krom] 字典/辭典 (dictionary)
  - 字構：พจน（言語/詞彙）+ านุกรม（排列整理，源自梵語）= 詞彙依序整理的書
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ค้น พจนานุกรม</span>（查字典）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">พจนานุกรม ไทย-อังกฤษ</span>（泰英字典）
  - 例：ไม่  รู้  ความ  หมาย  ของ  คำ  นี้  เลย  ต้อง  ค้น  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">พจนานุกรม</span>（不懂這個詞的意思，只好查字典）

- 14. **หนา** [nǎa] 厚 (thick)
  - 字構：หน้า（臉/頁，rising tone）vs หนา（厚，mid tone）→ 聲調一字之差，別記混
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">หนังสือ หนา</span>（厚書）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ผ้า หนา</span>（厚布料）
  - 例：หนังสือ  เล่ม  นี้  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">หนา</span>  มาก  กว่า  พัน  หน้า（這本書很厚，超過一千頁）

- 15. **การค้นคว้า** [kaan-khón-khwáa] 研究/查閱 (research)
  - 字構：ค้น（搜尋）+ คว้า（抓取）= 搜尋並抓取所需資訊
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">การ ค้นคว้า ข้อมูล</span>（查閱資料）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">งาน ค้นคว้า</span>（研究工作）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">การค้นคว้า</span>  จาก  แหล่ง  ที่  น่าเชื่อถือ  ช่วย  เพิ่ม  คุณภาพ  ของ  งาน（從可信來源研究有助於提升作品品質）

<a id="文化思想"></a>
## 主題五：文化思想

- 16. **ปรัชญา** [prà-chá-yaa] 哲學/理念 (philosophy)
  - 字構：มาจากสันสกฤต prajñā（智慧）→ 深刻智慧的學問
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปรัชญา ชีวิต</span>（人生哲學）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปรัชญา เศรษฐกิจ พอเพียง</span>（充足經濟哲學）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ปรัชญา</span>  เศรษฐกิจ  พอเพียง  สอน  ให้  เรา  รู้จัก  พอ（充足經濟哲學教導我們懂得知足）

- 17. **ลึกซึ้ง** [lʉ̂k-sʉ̂ng] 深刻/深邃 (profound/deeply)
  - 字構：ลึก（深）+ ซึ้ง（感動/觸動）= 深刻觸動心靈深處
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ความหมาย ลึกซึ้ง</span>（深刻的意義）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เข้าใจ อย่าง ลึกซึ้ง</span>（深刻理解）
  - 例：บทกวี  นี้  มี  ความ  หมาย  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ลึกซึ้ง</span>  กว่า  ที่  เห็น  ครั้งแรก（這首詩的含義比初看時更加深刻）

- 18. **มุมมอง** [mum-mɔɔng] 觀點/視角 (perspective/viewpoint)
  - 字構：มุม（角度/轉角）+ มอง（看）= 從某個角落的角度看事物
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">มุมมอง ต่างกัน</span>（不同觀點）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เปลี่ยน มุมมอง</span>（改變視角）
  - 例：ลอง  เปลี่ยน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">มุมมอง</span>  แล้ว  จะ  เข้าใจ  คน  อื่น  มากขึ้น（試著改變視角，就能更理解他人）

- 19. **สะท้อน** [sà-thɔ́ɔn] 反映/折射 (reflect)
  - 字構：สะ（反向前綴）+ ท้อน（彈回）= 像鏡面反射；引申為「反映」現象或想法
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">สะท้อน ปัญหา</span>（反映問題）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">สะท้อน สังคม</span>（反映社會）
  - 例：งาน  ศิลปะ  ชิ้น  นี้  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">สะท้อน</span>  ปัญหา  ความ  ไม่  เท่าเทียม  ได้  ดี（這件藝術作品很好地反映了不平等問題）

- 20. **เท่าเทียม** [thâo-thiam] 平等/均等 (equal/equitable)
  - 字構：เท่า（相等）+ เทียม（比肩/並排）= 並排且相等，不分高下
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ความ เท่าเทียม ทางเพศ</span>（性別平等）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">สังคม เท่าเทียม</span>（平等社會）
  - 例：ทุก  คน  ควร  มี  สิทธิ์  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เท่าเทียม</span>  กัน  ใน  สังคม（社會中每個人都應享有平等的權利）

- 21. **แบ่ง** [bɛ̀ng] 分享/分配 (share/divide)
  - 字構：เสียงสั้น → 一切為二，往兩側分開的動作
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">แบ่ง ปัน</span>（分享）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">แบ่ง งาน</span>（分配工作）
  - 例：เขา  ชอบ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">แบ่ง</span>  อาหาร  ให้  เพื่อน  ทุก  ครั้ง（他每次都喜歡把食物分享給朋友）

- 22. **งานศิลปะ** [ngaan-sǐn-lá-pà] 藝術作品 (artwork)
  - 字構：งาน（作品/工作）+ ศิลปะ（藝術，來自梵語 śilpa）= 藝術家的創作成果
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">งานศิลปะ ร่วมสมัย</span>（當代藝術）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">แสดง งานศิลปะ</span>（展示藝術作品）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">งานศิลปะ</span>  ชิ้น  นี้  ถูก  จัด  แสดง  ที่  หอ  ศิลป์  กรุงเทพ（這件藝術作品在曼谷藝術館展出）

<a id="動作日常"></a>
## 主題六：動作日常

- 23. **แขก** [khɛ̀ɛk] 客人/訪客 (guest/visitor)
  - 字構：諧音「客」→ 幾乎就是中文「客」的發音！
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ต้อนรับ แขก</span>（迎接客人）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">แขก มา เยี่ยม</span>（客人來拜訪）
  - 例：วันนี้  มี  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">แขก</span>  มา  เยี่ยม  บ้าน  หลาย  คน（今天有好幾位客人來家裡拜訪）

- 24. **ปู** [puu] 鋪/攤開 (to spread/lay out)
  - 字構：ปู = 螃蟹（同字雙義）→ 螃蟹橫向展開爪子，就像把東西鋪展開來
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปู โต๊ะ</span>（鋪桌）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปู ที่นอน</span>（鋪床）
  - 例：ช่วย  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ปู</span>  โต๊ะ  เพื่อ  ต้อนรับ  แขก  ด้วย  นะ（幫忙把桌子鋪好招待客人）

- 25. **กระเด็น** [krà-den] 彈開/飛濺 (bounce off/splatter)
  - 字構：กระ（動作強調前綴）+ เด็น（彈跳）= 被撞擊後快速彈射出去
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">น้ำ กระเด็น</span>（水花飛濺）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กระเด็น ออกไป</span>（彈飛出去）
  - 例：แก้ว  ตก  พื้น  แล้ว  เศษ  แก้ว  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กระเด็น</span>  ไป  ทั่ว（杯子摔到地上，玻璃碎片四處飛濺）

- 26. **ลอย** [lɔɔi] 漂浮/飄 (float)
  - 字構：諧音「囉依」→ 輕飄飄往上或往前漂移
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ลอย น้ำ</span>（漂在水上）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ลอย อยู่</span>（懸浮著）
  - 例：กลีบ  ดอกไม้  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ลอย</span>  อยู่  บน  ผิว  น้ำ  อย่าง  สวยงาม（花瓣優美地漂浮在水面上）

- 27. **แตก** [tɛ̀ɛk] 破裂/碎裂 (break/burst/crack)
  - 字構：เสียงสั้น กระทัน → 短促如破裂聲，像東西裂開的聲音
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">แตก ร้าว</span>（龜裂）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กระจก แตก</span>（玻璃破碎）
  - 例：กระจก  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">แตก</span>  ออก  เป็น  เสี่ยง  ๆ  หลังจาก  หิน  กระเด็น  มา  ชน（石頭彈來撞擊後，玻璃碎成一片片）
