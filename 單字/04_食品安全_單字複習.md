<a id="衛生安全"></a>
## 主題一：衛生安全

- 1. **ปลอดภัย** [plàwt phai] 安全 (safe)
  - 字構：ปลอด（脫離/免於）+ ภัย（危險）= 脫離危險
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">อาหาร ปลอดภัย</span>（安全食物）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ขับขี่ ปลอดภัย</span>（安全駕駛）
  - 例：กิน  อาหาร  ที่  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ปลอดภัย</span>  เพื่อ  ดูแล  สุขภาพ（吃安全的食物來照顧健康）

- 2. **อาหารเป็นพิษ** [aa hǎan pen phít] 食物中毒 (food poisoning)
  - 字構：อาหาร（食物）+ เป็น（是/變成）+ พิษ（毒）= 食物變成毒
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เสี่ยง อาหารเป็นพิษ</span>（有食物中毒風險）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ป้องกัน อาหารเป็นพิษ</span>（預防食物中毒）
  - 例：อากาศ  ร้อน  ทำให้  เสี่ยง  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">อาหารเป็นพิษ</span>  ได้  ง่าย（天氣熱讓食物中毒風險大增）

- 3. **เชื้อแบคทีเรีย** [chûea baek thii ria] 細菌 (bacteria)
  - 字構：เชื้อ（菌種）+ แบคทีเรีย（音譯 bacteria）
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เชื้อแบคทีเรีย ในอาหาร</span>（食物中的細菌）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เชื้อแบคทีเรีย เจริญเติบโต</span>（細菌繁殖）
  - 例：ความ  ร้อน  ทำให้  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เชื้อแบคทีเรีย</span>  เจริญเติบโต  ได้  เร็ว（高溫讓細菌繁殖得很快）

- 4. **เชื้อโรค** [chûea rôok] 病原體、病菌 (pathogen/germ)
  - 字構：เชื้อ（菌種）+ โรค（疾病）= 造成疾病的菌
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ทำลาย เชื้อโรค</span>（消滅病菌）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เชื้อโรค ปนเปื้อน</span>（病菌污染）
  - 例：ความ  ร้อน  สูง  สามารถ  ทำลาย  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เชื้อโรค</span>  ใน  อาหาร  ได้（高溫能消滅食物中的病菌）

- 5. **ปนเปื้อน** [pon pûean] 污染、沾染 (contaminate)
  - 字構：ปน（混入）+ เปื้อน（弄髒）= 混入髒東西
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">การปนเปื้อน ข้าม</span>（交叉污染）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ป้องกัน การปนเปื้อน</span>（預防污染）
  - 例：แยก  เขียง  ดิบ  สุก  เพื่อ  ป้องกัน  การ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ปนเปื้อน</span>  ข้าม（生熟分開砧板以預防交叉污染）

- 6. **สุขาภิบาล** [sù khǎa phi baan] 衛生管理、食品衛生 (sanitation)
  - 字構：สุขา（衛生/廁所）+ ภิบาล（管理照護）= 衛生管理
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">หลัก สุขาภิบาล</span>（衛生原則）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">สุขาภิบาล อาหาร</span>（食品衛生）
  - 例：ปรุง  อาหาร  ตาม  หลัก  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">สุขาภิบาล</span>  เพื่อ  ความ  ปลอดภัย（依衛生原則烹調食物確保安全）

- 7. **ป้องกัน** [pàwng kan] 預防、保護 (prevent/protect)
  - 字構：ป้อง（防護）+ กัน（互相/防止）= 相互防護
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ป้องกัน โรค</span>（預防疾病）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ป้องกัน ตัวเอง</span>（保護自己）
  - 例：ล้าง  มือ  ทุก  ครั้ง  ก่อน  กิน  เพื่อ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ป้องกัน</span>  โรค（每次飯前洗手以預防疾病）

<a id="烹調飲食"></a>
## 主題二：烹調飲食

- 8. **หลีกเลี่ยง** [lìik lîang] 避免、躲避 (avoid)
  - 字構：หลีก（偏開）+ เลี่ยง（繞過）= 繞開躲掉
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">หลีกเลี่ยง อาหารดิบ</span>（避免生食）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ควร หลีกเลี่ยง</span>（應該避免）
  - 例：ควร  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">หลีกเลี่ยง</span>  อาหาร  กึ่งสุก  ใน  ช่วง  อากาศ  ร้อน（天氣熱時應避免半熟食物）

- 9. **ปรุงสุก** [prung sùk] 煮熟 (cook until done)
  - 字構：ปรุง（烹調）+ สุก（熟）= 烹調到熟透
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปรุงสุก ทั่วถึง</span>（煮熟透徹）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ต้อง ปรุงสุก</span>（必須煮熟）
  - 例：อาหาร  ทะเล  ต้อง  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ปรุงสุก</span>  ทั่วถึง  ก่อน  รับประทาน（海鮮必須完全煮熟才能食用）

- 10. **กึ่งสุก** [kûeng sùk] 半熟 (semi-cooked)
  - 字構：กึ่ง（一半）+ สุก（熟）= 只熟了一半
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">อาหาร กึ่งสุก</span>（半熟食物）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เลี่ยง กึ่งสุก</span>（避免半熟）
  - 例：อย่า  กิน  อาหาร  ทะเล  แบบ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กึ่งสุก</span>  เพราะ  เชื้อโรค  ยัง  อยู่（不要吃半熟海鮮因為病菌還殘留）

- 11. **อุณหภูมิ** [ùn hà phuum] 溫度 (temperature)
  - 字構：來自梵語，諧音「溫哈普」→ 溫度讓人哈氣
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">อุณหภูมิ 70 องศา</span>（70度溫度）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตั้ง อุณหภูมิ</span>（設定溫度）
  - 例：ปรุง  อาหาร  ที่  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">อุณหภูมิ</span>  70  องศา  ขึ้นไป  เพื่อ  ฆ่า  เชื้อโรค（在70度以上烹調以殺滅病菌）

- 12. **เจริญเติบโต** [jà roen tòep too] 生長、繁殖 (grow/proliferate)
  - 字構：เจริญ（繁盛）+ เติบโต（長大）= 蓬勃生長
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เชื้อโรค เจริญเติบโต</span>（病菌繁殖）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เจริญเติบโต เร็ว</span>（快速增長）
  - 例：ความ  ชื้น  และ  ความ  ร้อน  ช่วย  ให้  เชื้อโรค  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เจริญเติบโต</span>  เร็ว（濕熱有助於病菌快速繁殖）

- 13. **ล้างมือ** [láang muue] 洗手 (wash hands)
  - 字構：ล้าง（清洗）+ มือ（手）= 洗手
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ล้างมือ ก่อนกิน</span>（飯前洗手）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ล้างมือ ด้วยสบู่</span>（用肥皂洗手）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ล้างมือ</span>  ด้วย  สบู่  ทุก  ครั้ง  หลัง  ใช้  ห้องน้ำ（每次上廁所後用肥皂洗手）

## 主題三：食物種類

- 14. **อาหารทะเล** [aa hǎan thá lee] 海鮮 (seafood)
  - 字構：อาหาร（食物）+ ทะเล（海）= 海裡的食物
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">อาหารทะเล สด</span>（新鮮海鮮）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปรุง อาหารทะเล</span>（烹調海鮮）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">อาหารทะเล</span>  ต้อง  ปรุง  ให้  สุก  ทันที  หลัง  ซื้อ（海鮮買來後要馬上煮熟）

- 15. **อาหารดิบ** [aa hǎan dìp] 生食 (raw food)
  - 字構：อาหาร（食物）+ ดิบ（生的/未熟）= 未煮的食物
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">หลีกเลี่ยง อาหารดิบ</span>（避免生食）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">อาหารดิบ อันตราย</span>（生食危險）
  - 例：ใน  อากาศ  ร้อน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">อาหารดิบ</span>  เสีย  ได้  เร็ว  มาก（天氣熱時生食壞得非常快）

- 16. **ของดอง** [khǎawng dawng] 醃漬食品 (fermented/pickled food)
  - 字構：ของ（東西）+ ดอง（醃漬）= 醃漬的東西
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ของดอง ดิบ</span>（生醃食品）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปูม้า ดอง</span>（醃漬梭子蟹）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ของดอง</span>  ที่  ไม่  ผ่าน  ความ  ร้อน  อาจ  มี  เชื้อโรค（未經加熱的醃漬食品可能帶有病菌）

- 17. **น้ำสะอาด** [náam sà àat] 乾淨的水 (clean water)
  - 字構：น้ำ（水）+ สะอาด（乾淨）= 乾淨的水
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ใช้ น้ำสะอาด</span>（使用乾淨的水）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">น้ำสะอาด ล้างผัก</span>（用乾淨的水洗蔬菜）
  - 例：ควร  ใช้  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">น้ำสะอาด</span>  ในการ  เล่น  สงกรานต์  เพื่อ  ป้องกัน  โรค  ผิวหนัง（宋干節應使用乾淨的水以預防皮膚病）

<a id="症狀與處理"></a>
## 主題四：症狀與處理

- 18. **ปวดท้อง** [pùat tháwng] 肚子痛 (stomach ache)
  - 字構：ปวด（疼痛）+ ท้อง（肚子）= 肚子疼
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">มีอาการ ปวดท้อง</span>（有腹痛症狀）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปวดท้อง ท้องเสีย</span>（腹痛腹瀉）
  - 例：หลัง  กิน  อาหาร  ทะเล  ดิบ  มี  อาการ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ปวดท้อง</span>  ทันที（吃了生海鮮後馬上腹痛）

- 19. **ท้องเสีย** [tháwng sǐa] 腹瀉 (diarrhea)
  - 字構：ท้อง（肚子）+ เสีย（壞掉/出問題）= 肚子出問題
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">มีอาการ ท้องเสีย</span>（有腹瀉症狀）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ท้องเสีย รุนแรง</span>（嚴重腹瀉）
  - 例：หาก  มีอาการ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ท้องเสีย</span>  ต้อง  ดื่ม  น้ำ  เกลือ  แร่  ทดแทน（如果腹瀉要喝電解質水補充）

- 20. **อาเจียน** [aa jian] 嘔吐 (vomit)
  - 字構：諧音「阿見」→ 嘔吐看到了什麼
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">คลื่นไส้ อาเจียน</span>（噁心嘔吐）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">อาเจียน หลาย ครั้ง</span>（多次嘔吐）
  - 例：มีอาการ  คลื่นไส้  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">อาเจียน</span>  หลัง  กิน  อาหาร  เป็น  พิษ（食物中毒後出現噁心嘔吐）

- 21. **ส่งผลให้** [sòng phǒn hâi] 導致、造成 (result in/cause)
  - 字構：ส่ง（傳送）+ ผล（結果）+ ให้（給）= 傳送結果給某事
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ส่งผลให้ เสี่ยง</span>（導致風險）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ส่งผลให้ เกิดโรค</span>（導致生病）
  - 例：อากาศ  ร้อน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ส่งผลให้</span>  เชื้อโรค  เจริญเติบโต  เร็ว（炎熱天氣導致病菌快速繁殖）

<a id="句型整理"></a>
## 句型整理

### 1. ควร + กริยา — 應該…（建議）

| 泰文 | 中文 |
|------|------|
| ควร  หลีกเลี่ยง  อาหารดิบ | 應該避免生食 |
| ควร  ล้างมือ  ก่อน  ปรุงอาหาร | 應該在烹調前洗手 |
| ควร  ใช้  น้ำสะอาด  ในการ  ปรุง | 應該用乾淨的水烹調 |

---

### 2. ต้อง + กริยา — 必須…（義務）

| 泰文 | 中文 |
|------|------|
| ต้อง  ปรุงสุก  ทั่วถึง | 必須完全煮熟 |
| ต้อง  ดื่ม  น้ำ  เกลือ  แร่ | 必須喝電解質水 |
| ต้อง  ไป  พบ  แพทย์  ทันที | 必須立刻去看醫生 |

---

### 3. หลีกเลี่ยง + นาม — 避免…

| 泰文 | 中文 |
|------|------|
| หลีกเลี่ยง  อาหาร  กึ่งสุก | 避免半熟食物 |
| หลีกเลี่ยง  ของ  ดอง  ดิบ | 避免生醃食品 |
| หลีกเลี่ยง  การ  ปนเปื้อน  ข้าม | 避免交叉污染 |

---

### 4. เพื่อ + กริยา — 為了…（目的）

| 泰文 | 中文 |
|------|------|
| ล้าง  มือ  เพื่อ  ป้องกัน  เชื้อโรค | 洗手以預防病菌 |
| ปรุงสุก  เพื่อ  ทำลาย  แบคทีเรีย | 煮熟以消滅細菌 |
| แยก  เขียง  เพื่อ  ป้องกัน  การ  ปนเปื้อน | 分開砧板以預防污染 |

---

### 5. หาก + ประโยค — 如果…（條件句）

| 泰文 | 中文 |
|------|------|
