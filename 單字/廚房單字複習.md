<a id="餐具"></a>
## 主題一：餐具

- 1. **มีด** [mîit] 刀 (knife)
  - 字構：諧音「咪」→ 咪咪貓拿刀切魚
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">มีด ครัว</span>（廚刀）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">มีด คม</span>（刀很鋒利）
  - 例：ระวัง  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">มีด</span>  คม  อยู่（小心，刀很鋒利）

- 2. **ส้อม** [sôm] 叉子 (fork)
  - 字構：諧音「頌」→ 頌揚叉子的發明
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ช้อน ส้อม</span>（湯匙和叉子）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ส้อม จิ้ม</span>（用叉子叉）
  - 例：ใช้  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ส้อม</span>  จิ้ม  เนื้อ  ได้  เลย（可以直接用叉子叉肉）

- 3. **ช้อน** [chówn] 湯匙 (spoon)
  - 字構：諧音「稱」→ 用湯匙稱量份量
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ช้อน โต๊ะ</span>（餐匙）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ช้อน ชา</span>（茶匙）
  - 例：ใส่  เกลือ  หนึ่ง  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ช้อน</span>  ชา（加一茶匙鹽）

- 4. **ตะเกียบ** [tà-kìap] 筷子 (chopsticks)
  - 字構：諧音「打跌巴」→ 學筷子夾東西一直打跌
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ใช้ ตะเกียบ</span>（使用筷子）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตะเกียบ ไม้</span>（木筷子）
  - 例：คน  จีน  กิน  ข้าว  ด้วย  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ตะเกียบ</span>（中國人用筷子吃飯）

- 5. **จาน** [jaan] 盤子 (plate)
  - 字構：諧音「站」→ 食物站在盤子上
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">จาน ข้าว</span>（飯盤）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ล้าง จาน</span>（洗盤子）
  - 例：ช่วย  ล้าง  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">จาน</span>  ด้วย  นะ（幫忙洗盤子好嗎）

- 6. **ชาม** [chaam] 碗 (bowl)
  - 字構：諧音「查姆」→ 查查碗裡有什麼
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ชาม ข้าว</span>（飯碗）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ชาม ก๋วยเตี๋ยว</span>（麵碗）
  - 例：เอา  ก๋วยเตี๋ยว  ใส่  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ชาม</span>  ให้  ด้วย（幫我把麵放進碗裡）

- 7. **แก้ว** [gâew] 玻璃杯 (glass/cup)
  - 字構：諧音「給」→ 給我一杯水
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">แก้ว น้ำ</span>（水杯）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">แก้ว เบียร์</span>（啤酒杯）
  - 例：เติม  น้ำ  ใน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">แก้ว</span>  ให้  หน่อย（幫我在杯子裡加點水）

- 8. **หม้อ** [mâw] 鍋子 (pot)
  - 字構：諧音「冒」→ 鍋子冒出熱氣
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">หม้อ ต้ม</span>（燉鍋）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">หม้อ ใบ ใหญ่</span>（大鍋子）
  - 例：ต้ม  น้ำ  ใน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">หม้อ</span>  จน  เดือด（在鍋子裡把水煮到滾）

- 9. **กระทะ** [grà-thá] 炒鍋/平底鍋 (wok/pan)
  - 字構：諧音「葛踏」→ 葛師傅踏步炒鍋
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กระทะ ผัด</span>（炒菜鍋）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตั้ง กระทะ</span>（起鍋/放上鍋子）
  - 例：ตั้ง  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กระทะ</span>  แล้ว  ใส่  น้ำมัน  ลงไป（起鍋後加入油）

- 10. **เขียง** [khǐang] 砧板 (cutting board)
  - 字構：諧音「蟹昂」→ 蟹昂首挺胸站在砧板上
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เขียง ไม้</span>（木砧板）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">หั่น บน เขียง</span>（在砧板上切）
  - 例：หั่น  ผัก  บน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เขียง</span>  ก่อน  นะ（先在砧板上把蔬菜切好）

- 11. **ถ้วย** [thûay] 小杯/茶杯 (cup)
  - 字構：諧音「推」→ 推過去一杯熱茶
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ถ้วย ชา</span>（茶杯）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ถ้วย กาแฟ</span>（咖啡杯）
  - 例：เทน้ำชา  ใส่  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ถ้วย</span>  ให้  ผู้ใหญ่（把茶倒入杯子給長輩）

<a id="廚房設備"></a>
## 主題二：廚房設備

- 12. **เตา** [tao] 爐子 (stove)
  - 字構：諧音「刀」→ 爐子旁邊放著刀
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เตา แก๊ส</span>（瓦斯爐）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตั้ง ไฟ บน เตา</span>（在爐上開火）
  - 例：เปิด  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เตา</span>  แก๊ส  แล้ว  ตั้ง  หม้อ（打開瓦斯爐然後放上鍋子）

- 13. **เตาอบ** [tao-òp] 烤箱 (oven)
  - 字構：เตา（爐）+ อบ（烘烤）= 烘烤的爐
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">อบ ใน เตาอบ</span>（放入烤箱烤）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตั้ง อุณหภูมิ เตาอบ</span>（設定烤箱溫度）
  - 例：อบ  ขนมปัง  ใน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เตาอบ</span>  180  องศา（用180度烤箱烤麵包）

- 14. **ตู้เย็น** [tûu-yen] 冰箱 (refrigerator)
  - 字構：ตู้（櫃子）+ เย็น（冷）= 冷的櫃子
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เก็บ ใน ตู้เย็น</span>（放進冰箱）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตู้เย็น เต็ม</span>（冰箱滿了）
  - 例：เก็บ  อาหาร  เหลือ  ใน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ตู้เย็น</span>  ได้  เลย（剩下的食物可以直接放冰箱）

- 15. **ไมโครเวฟ** [mai-khro-wéep] 微波爐 (microwave)
  - 記憶法：直接音譯自英文 microwave
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">อุ่น ใน ไมโครเวฟ</span>（用微波爐加熱）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ไมโครเวฟ 2 นาที</span>（微波2分鐘）
  - 例：อุ่น  ข้าว  ใน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ไมโครเวฟ</span>  สอง  นาที（把飯放微波爐加熱兩分鐘）

- 16. **หม้อหุงข้าว** [mâw-hǔng-khâao] 電鍋 (rice cooker)
  - 字構：หม้อ（鍋）+ หุง（煮/炊）+ ข้าว（飯）= 煮飯的鍋
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">หุง ข้าว ใน หม้อหุงข้าว</span>（用電鍋煮飯）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">หม้อหุงข้าว ไฟฟ้า</span>（電飯鍋）
  - 例：ซื้อ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">หม้อหุงข้าว</span>  ไฟฟ้า  ตัว  ใหม่（買了一台新電飯鍋）

- 17. **เครื่องปั่น** [khrûeang-pân] 果汁機/攪拌機 (blender)
  - 字構：เครื่อง（機器）+ ปั่น（旋轉攪打）= 旋轉攪打的機器
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปั่น ใน เครื่องปั่น</span>（用果汁機打）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เครื่องปั่น น้ำผลไม้</span>（果汁機）
  - 例：ปั่น  ผลไม้  กับ  นม  รวมกัน（把水果和牛奶一起打）

- 18. **กระชอน** [grà-chawn] 濾網/漏杓 (strainer/colander)
  - 字構：諧音「葛叉恩」→ 葛師傅用叉子恩賜濾網
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กรอง ด้วย กระชอน</span>（用濾網過濾）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กระชอน ตาถี่</span>（細網濾網）
  - 例：กรอง  น้ำซุป  ผ่าน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กระชอน</span>  ก่อน（先用濾網過濾湯汁）

- 56. **ช้อนไม้** [cháwn mái] 木匙 (wooden spoon)
  - 字構：ช้อน（湯匙）+ ไม้（木）= 木製湯匙
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ช้อนไม้ คน</span>（用木匙攪拌）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ช้อนไม้ ยาว</span>（長木匙）
  - 例：คน  ซุป  ด้วย  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ช้อนไม้</span>  ช้าๆ（用木匙慢慢攪拌湯）

- 57. **กระทะเคลือบ** [krà-thá-khlûeap] 不沾鍋 (non-stick pan)
  - 字構：กระทะ（鍋）+ เคลือบ（塗層）= 有不沾塗層的鍋
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ทอด ใน กระทะเคลือบ</span>（用不沾鍋煎）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ไข่ ไม่ ติด กระทะ</span>（蛋不沾鍋）
  - 例：ทอด  ไข่  ใน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กระทะเคลือบ</span>  โดย  ไม่  ต้อง  ใช้  น้ำมัน  มาก（用不沾鍋煎蛋不需要太多油）

- 58. **ถ้วยตวง** [thûay-tuang] 量杯 (measuring cup)
  - 字構：ถ้วย（杯）+ ตวง（量取）= 用來量取的杯子
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตวง แป้ง ด้วย ถ้วยตวง</span>（用量杯量麵粉）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ถ้วยตวง หนึ่ง ถ้วย</span>（一量杯）
  - 例：ตวง  แป้ง  สอง  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ถ้วยตวง</span>  สำหรับ  ทำ  เค้ก（量兩量杯麵粉來做蛋糕）

- 59. **ช้อนตวง** [cháwn-tuang] 量匙 (measuring spoon)
  - 字構：ช้อน（匙）+ ตวง（量取）= 用來量取的匙
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ช้อนตวง ชา</span>（茶匙量匙）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตวง ด้วย ช้อนตวง</span>（用量匙量取）
  - 例：ใส่  ผง  ฟู  หนึ่ง  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ช้อนตวง</span>  ลง  ใน  แป้ง（把一量匙泡打粉加入麵粉中）

- 60. **ที่รองหม้อ** [thîi-rawng-mâw] 鍋墊/隔熱墊 (trivet/pot holder)
  - 字構：ที่（處所）+ รอง（墊在下面）+ หม้อ（鍋）= 墊在熱鍋下的隔熱物
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">วาง บน ที่รองหม้อ</span>（放在鍋墊上）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ที่รองหม้อ กันความร้อน</span>（隔熱鍋墊）
  - 例：วาง  หม้อ  ร้อน  บน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ที่รองหม้อ</span>  เสมอ（要隨時把熱鍋放在鍋墊上）

- 61. **ตะกร้อมือ** [tà-kráw-muue] 手動打蛋器 (hand whisk)
  - 字構：ตะกร้อ（球形格架）+ มือ（手）= 用手操作的球形打蛋器
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตี ไข่ ด้วย ตะกร้อมือ</span>（用打蛋器打蛋）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตะกร้อมือ สแตนเลส</span>（不鏽鋼打蛋器）
  - 例：ตี  ไข่  กับ  นม  ด้วย  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ตะกร้อมือ</span>  จน  เนื้อ  เนียน（用打蛋器把蛋和牛奶打到滑順）

- 62. **ลังถึง** [lang-thûeng] 蒸籠/蒸架 (steamer basket)
  - 字構：ลัง（籠架）+ ถึง（透達）= 讓蒸氣透達的蒸籠
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">นึ่ง ใน ลังถึง</span>（在蒸籠裡蒸）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ลังถึง ไม้ไผ่</span>（竹製蒸籠）
  - 例：นึ่ง  ขนมจีบ  ใน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ลังถึง</span>  10  นาที（把燒賣放蒸籠蒸10分鐘）

- 63. **เครื่องปิ้งขนมปัง** [khrûeang-pîng-khanǒm-pang] 烤麵包機 (toaster)
  - 字構：เครื่อง（機器）+ ปิ้ง（烤）+ ขนมปัง（麵包）= 烤麵包的機器
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปิ้ง ขนมปัง ใน เครื่องปิ้งขนมปัง</span>（用烤麵包機烤）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เครื่องปิ้งขนมปัง ไฟฟ้า</span>（電烤麵包機）
  - 例：ปิ้ง  ขนมปัง  ใน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เครื่องปิ้งขนมปัง</span>  ระหว่าง  ชง  กาแฟ（一邊等咖啡一邊用烤麵包機烤麵包）

- 64. **อ่างล้างจาน** [àang-láang-jaan] 洗碗槽 (kitchen sink)
  - 字構：อ่าง（盆槽）+ ล้าง（清洗）+ จาน（盤子）= 清洗盤子的槽
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ล้าง จาน ใน อ่างล้างจาน</span>（在洗碗槽洗盤子）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">อ่างล้างจาน สองช่อง</span>（雙槽洗碗槽）
  - 例：นำ  จาน  ทั้งหมด  ไป  ล้าง  ใน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">อ่างล้างจาน</span>（把所有盤子拿去洗碗槽洗）

- 65. **ตะแกรง** [tà-kraeng] 篩網/細篩 (fine sieve)
  - 字構：諧音「踏拐恩」→ 踏過篩網過濾食材留下精華
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กรอง ผ่าน ตะแกรง</span>（透過篩網過濾）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตะแกรง ตาถี่</span>（密目篩網）
  - 例：กรอง  ซอส  ผ่าน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ตะแกรง</span>  เพื่อ  เอา  กาก  ออก（把醬汁透過篩網過濾除去殘渣）

- 66. **กาต้มน้ำ** [kaa-tôm-náam] 煮水壺/電熱壺 (kettle)
  - 字構：กา（壺）+ ต้ม（煮沸）+ น้ำ（水）= 煮水的壺
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ต้ม น้ำ ด้วย กาต้มน้ำ</span>（用水壺煮水）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กาต้มน้ำ ไฟฟ้า</span>（電熱水壺）
  - 例：ต้ม  น้ำ  ใน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กาต้มน้ำ</span>  แล้ว  ชง  ชา  ใน  ตอน  เช้า（用水壺煮水後在早晨泡茶）

- 67. **กล่องปิดสนิท** [klàwng-pìt-sà-nìt] 密封盒 (airtight container)
  - 字構：กล่อง（盒子）+ ปิด（關閉）+ สนิท（緊密）= 緊密關閉的盒子
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เก็บ อาหาร ใน กล่องปิดสนิท</span>（把食物放入密封盒）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กล่องปิดสนิท พลาสติก</span>（塑膠密封盒）
  - 例：เก็บ  อาหาร  ที่  เหลือ  ไว้  ใน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กล่องปิดสนิท</span>  ใน  ตู้เย็น（把剩餘的食物放入密封盒存放在冰箱）

- 68. **เครื่องเตรียมอาหาร** [khrûeang-triam-aa-hǎan] 食物調理機 (food processor)
  - 字構：เครื่อง（機器）+ เตรียม（準備）+ อาหาร（食物）= 準備食物的機器
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ใช้ เครื่องเตรียมอาหาร หั่น ผัก</span>（用調理機切菜）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เครื่องเตรียมอาหาร หลายฟังก์ชัน</span>（多功能調理機）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เครื่องเตรียมอาหาร</span>  ช่วย  ประหยัด  เวลา  ใน  การ  หั่น  ผัก（食物調理機幫助節省切菜時間）

<a id="料理方式"></a>
## 主題三：料理方式

- 19. **ทอด** [thâwt] 油炸 (deep fry)
  - 字構：諧音「托」→ 托著油炸物
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ทอด ใน น้ำมัน</span>（在油裡炸）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ไข่ ดาว</span>（煎蛋/荷包蛋）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ทอด</span>  ปลา  ใน  น้ำมัน  ร้อน（在熱油裡炸魚）

- 20. **ผัด** [phàt] 炒 (stir-fry)
  - 字構：諧音「拍」→ 拍打鍋子快速翻炒
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ผัด ผัก</span>（炒蔬菜）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ผัด ไฟ แรง</span>（大火快炒）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ผัด</span>  ผัก  ไฟ  แรง  ให้  หอม（大火快炒蔬菜讓它香）

- 21. **ต้ม** [tôm] 水煮/燉 (boil)
  - 字構：諧音「冬」→ 冬天喝水煮湯暖身
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ต้ม น้ำ</span>（煮水）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ต้ม จน สุก</span>（煮到熟）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ต้ม</span>  น้ำ  จน  เดือด  แล้ว  ใส่  เส้น（水煮滾後放麵條）

- 22. **นึ่ง** [nûeng] 蒸 (steam)
  - 字構：諧音「能」→ 能量透過蒸氣傳導
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">นึ่ง ปลา</span>（蒸魚）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">นึ่ง ใน ซึ้ง</span>（用蒸籠蒸）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">นึ่ง</span>  ปลา  กับ  ขิง  และ  ต้นหอม（蒸魚配薑和蔥）

- 23. **ย่าง** [yâang] 燒烤 (grill/roast)
  - 字構：諧音「樣」→ 各式各樣的燒烤
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ย่าง ไก่</span>（烤雞）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ย่าง บน เตา</span>（在烤架上烤）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ย่าง</span>  หมู  บน  เตา  ถ่าน（在炭火上烤豬肉）

- 24. **อบ** [òp] 烘烤（箱烤）(bake)
  - 字構：諧音「喔」→ 喔！烤箱烤好了
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">อบ ขนม</span>（烤糕點）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">อบ ด้วย เตาอบ</span>（用烤箱烤）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">อบ</span>  คุกกี้  ที่  อุณหภูมิ  160  องศา（用160度烤餅乾）

- 25. **ตุ๋น** [tǔn] 燉/慢燉 (stew/braise)
  - 字構：諧音「頓」→ 頓一頓慢慢燉
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตุ๋น เนื้อ</span>（燉牛肉）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตุ๋น ไฟ อ่อน</span>（小火慢燉）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ตุ๋น</span>  เนื้อ  ไฟ  อ่อน  สอง  ชั่วโมง（小火燉牛肉兩小時）

- 26. **คั่ว** [khûa] 乾炒/煸 (dry roast/toast)
  - 字構：諧音「夸」→ 夸獎乾炒出來的香氣
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">คั่ว พริก</span>（乾炒辣椒）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">คั่ว ถั่ว</span>（炒花生）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">คั่ว</span>  ถั่ว  ลิสง  ให้  หอม  กรอบ（把花生乾炒到香脆）

- 27. **หมัก** [màk] 醃製/醃漬 (marinate)
  - 字構：諧音「麥」→ 麥子泡在水裡醃漬發酵
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">หมัก เนื้อ</span>（醃肉）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">หมัก ไว้ ข้ามคืน</span>（醃一夜）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">หมัก</span>  ไก่  กับ  ซีอิ๊ว  ไว้  หนึ่ง  ชั่วโมง（把雞肉用醬油醃一小時）

- 28. **ปรุง รส** [prung rót] 調味 (season/flavor)
  - 字構：ปรุง（料理/調配）+ รส（味道）= 調配味道
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปรุง รส ให้ อร่อย</span>（調味讓它好吃）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปรุง ตาม ชอบ</span>（依喜好調味）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ปรุง  รส</span>  ด้วย  น้ำปลา  และ  น้ำตาล（用魚露和糖調味）

- 69. **ตวง** [tuang] 量取/稱量（食材）(measure ingredients)
  - 字構：諧音「頂」→ 頂到刻度線就是量好了
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตวง แป้ง</span>（量麵粉）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตวง อย่าง ระมัดระวัง</span>（小心量取）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ตวง</span>  น้ำตาล  สอง  ช้อนโต๊ะ  ใส่  ใน  ชาม（量兩湯匙糖放入碗中）

- 70. **ชง** [chong] 沖泡/沖煮 (brew/steep)
  - 字構：諧音「充」→ 充入熱水沖泡出味道
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ชง ชา</span>（泡茶）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ชง กาแฟ</span>（煮咖啡）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ชง</span>  กาแฟ  ด้วย  น้ำ  ร้อน  ใน  ตอน  เช้า（早晨用熱水沖煮咖啡）

- 71. **กรอง** [krawng] 過濾/濾除 (strain/filter)
  - 字構：諧音「公」→ 公道地過濾留下精華去掉雜質
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กรอง น้ำซุป</span>（過濾湯汁）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กรอง เอา กาก ออก</span>（過濾除去殘渣）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กรอง</span>  น้ำซุป  ให้  ใส  ก่อน  เสิร์ฟ（上菜前把湯汁過濾清澈）

- 72. **เช็ด** [chét] 擦乾/擦拭 (wipe/dry)
  - 字構：諧音「扯」→ 扯布把水分和污垢擦乾
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เช็ด ให้ แห้ง</span>（擦乾）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เช็ด ด้วย ผ้า</span>（用布擦）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เช็ด</span>  จาน  ให้  แห้ง  หลังจาก  ล้าง  แล้ว（洗完盤子後把它擦乾）

- 73. **อุ่น** [ùn] 加熱/溫熱 (reheat/warm up)
  - 字構：諧音「溫」→ 溫熱起來讓食物回溫
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">อุ่น อาหาร</span>（加熱食物）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">อุ่น ใน ไมโครเวฟ</span>（用微波爐加熱）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">อุ่น</span>  ซุป  ใน  หม้อ  เล็ก  ด้วย  ไฟ  อ่อน（用小鍋小火溫熱湯）

<a id="食材處理"></a>
## 主題四：食材處理

- 29. **หั่น** [hàn] 切/切片 (slice/cut)
  - 字構：諧音「漢」→ 漢子拿刀切食材
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">หั่น เป็น ชิ้น</span>（切成塊）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">หั่น บาง ๆ</span>（切薄片）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">หั่น</span>  หมู  เป็น  ชิ้น  บาง ๆ（把豬肉切成薄片）

- 30. **สับ** [sàp] 剁/切碎 (chop/mince)
  - 字構：諧音「沙布」→ 沙布上剁肉碎
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">สับ เนื้อ</span>（剁碎肉）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">สับ กระเทียม</span>（剁蒜末）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">สับ</span>  กระเทียม  ให้  ละเอียด（把大蒜剁成細末）

- 31. **ปอก** [pàwk] 削皮 (peel)
  - 字構：諧音「剝」→ 剝皮（台語）
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปอก เปลือก</span>（削皮）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปอก ส้ม</span>（剝橘子皮）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ปอก</span>  เปลือก  มะม่วง  ก่อน  กิน（吃芒果前先削皮）

- 32. **ล้าง** [láang] 清洗 (wash)
  - 字構：諧音「浪」→ 用水浪沖洗乾淨
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ล้าง ผัก</span>（洗蔬菜）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ล้าง มือ</span>（洗手）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ล้าง</span>  ผัก  ให้  สะอาด  ก่อน  หั่น（切之前先把蔬菜洗乾淨）

- 33. **คน** [khon] 攪拌 (stir)
  - 記憶法：คน 本意是「人」→ 用人的力量攪拌
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">คน ให้ เข้ากัน</span>（攪拌均勻）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">คน ตลอด</span>（持續攪拌）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">คน</span>  ซุป  ตลอด  เวลา  เพื่อ  ไม่  ให้  ติด  ก้น  หม้อ（持續攪拌湯避免黏鍋底）

- 34. **ตี** [tii] 打（蛋）/打發 (beat/whisk)
  - 字構：諧音「踢」→ 踢打蛋讓它起泡
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตี ไข่</span>（打蛋）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตี ให้ ฟู</span>（打到蓬鬆）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ตี</span>  ไข่  ให้  เข้ากัน  ก่อน  ใส่  กระทะ（把蛋打勻再放入鍋中）

- 35. **บด** [bòt] 研磨/搗碎 (grind/pound)
  - 字構：諧音「剝」→ 剝開後再磨碎
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">บด พริก</span>（磨辣椒）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">บด ให้ ละเอียด</span>（磨到細緻）
  - 例：<span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">บด</span>  เนื้อ  วัว  ให้  ละเอียด  สำหรับ  ทำ  เบอร์เกอร์（把牛肉磨碎做漢堡）

- 74. **คม** [khom] 鋒利（形容詞）(sharp)
  - 字構：諧音「控」→ 要控制好鋒利的刀才安全
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">มีด คม</span>（鋒利的刀）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">คม มาก</span>（非常鋒利）
  - 例：ระวัง  มีด  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">คม</span>  อยู่  บน  เขียง（小心砧板上有鋒利的刀）

- 75. **ระมัดระวัง** [rá-màt-rá-wang] 小心謹慎（副詞）(carefully/cautious)
  - 字構：ระมัด（謹慎）+ ระวัง（小心注意）= 謹慎小心
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ทำ อย่าง ระมัดระวัง</span>（謹慎地做）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ระมัดระวัง เรื่อง ความ ปลอดภัย</span>（謹慎注意安全）
  - 例：ตวง  แป้ง  อย่าง  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ระมัดระวัง</span>  เพื่อ  ให้  ได้  ปริมาณ  ที่  ถูก  ต้อง（謹慎量取麵粉以獲得正確份量）

- 76. **อัตโนมัติ** [àt-tà-noo-mát] 自動（形容詞）(automatic)
  - 字構：來自梵語，諧音「阿踏諾馬提」→ 自動運作不需動手操作
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปิด เอง อัตโนมัติ</span>（自動關閉）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ระบบ อัตโนมัติ</span>（自動系統）
  - 例：หม้อหุงข้าว  จะ  ปิด  เอง  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">อัตโนมัติ</span>  เมื่อ  ข้าว  สุก（電鍋在飯熟後會自動關閉）

- 77. **เนื้อเนียน** [núea-nian] 質地細滑/滑順 (smooth texture)
  - 字構：เนื้อ（肉/質地）+ เนียน（滑順）= 滑順的質地
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตี จน เนื้อเนียน</span>（打到質地細滑）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เนื้อ เนียน ดี</span>（質地很細滑）
  - 例：ตี  ครีม  จน  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เนื้อเนียน</span>  และ  ฟู（把奶油打到細滑蓬鬆）

<a id="調味料"></a>
## 主題五：調味料

- 36. **เกลือ** [kloea] 鹽 (salt)
  - 字構：諧音「嘎了」→ 嘎！加了太多鹽了
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ใส่ เกลือ</span>（加鹽）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เกลือ นิดหน่อย</span>（一點點鹽）
  - 例：ใส่  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เกลือ</span>  นิดหน่อย  เพื่อ  เพิ่ม  รสชาติ（加一點鹽提升風味）

- 37. **น้ำตาล** [náam-taan] 糖 (sugar)
  - 字構：น้ำ（水/液體）+ ตาล（棕櫚樹）= 棕櫚樹的汁液→ 糖
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ใส่ น้ำตาล</span>（加糖）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">น้ำตาล ทราย</span>（砂糖）
  - 例：ใส่  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">น้ำตาล</span>  ทราย  หนึ่ง  ช้อน  โต๊ะ（加一湯匙砂糖）

- 38. **น้ำปลา** [náam-plaa] 魚露 (fish sauce)
  - 字構：น้ำ（水/液體）+ ปลา（魚）= 魚的液體
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ปรุง ด้วย น้ำปลา</span>（用魚露調味）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">น้ำปลา หนึ่ง ช้อน</span>（一匙魚露）
  - 例：อาหาร  ไทย  ส่วนใหญ่  ใช้  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">น้ำปลา</span>  แทน  เกลือ（泰國食物大多用魚露代替鹽）

- 39. **ซีอิ๊ว** [sii-íw] 醬油 (soy sauce)
  - 記憶法：來自潮州話「豉油」的音譯
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ซีอิ๊ว ขาว</span>（淡醬油）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ซีอิ๊ว ดำ</span>（黑醬油/老抽）
  - 例：ใส่  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ซีอิ๊ว</span>  ขาว  และ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ซีอิ๊ว</span>  ดำ  ลงไป  ผัด（加入淡醬油和老抽一起炒）

- 40. **น้ำมัน** [náam-man] 油 (oil)
  - 字構：น้ำ（液體）+ มัน（油脂）= 油脂液體
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">น้ำมัน พืช</span>（植物油）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ใส่ น้ำมัน ลง กระทะ</span>（把油倒入鍋中）
  - 例：ใส่  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">น้ำมัน</span>  ลง  กระทะ  แล้ว  รอ  ให้  ร้อน（把油倒入鍋中等熱）

- 41. **กระเทียม** [grà-thiam] 大蒜 (garlic)
  - 字構：諧音「葛提姆」→ 葛師傅提著蒜頭
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">สับ กระเทียม</span>（剁蒜末）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เจียว กระเทียม</span>（爆香蒜頭）
  - 例：เจียว  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กระเทียม</span>  ให้  หอม  ก่อน  ผัด  ผัก（炒蔬菜前先爆香蒜頭）

- 42. **พริก** [phrík] 辣椒 (chili)
  - 字構：諧音「批力」→ 批力（辣椒）讓人很有力
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">พริก เผ็ด</span>（辣的辣椒）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ใส่ พริก</span>（加辣椒）
  - 例：คน  ไทย  ชอบ  ใส่  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">พริก</span>  ในอาหาร  ทุก  อย่าง（泰國人喜歡在每道菜裡加辣椒）

- 43. **พริกไทย** [phrík-thai] 胡椒 (pepper)
  - 字構：พริก（辣椒/辛香料）+ ไทย（泰國）= 泰國辛香料
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">พริกไทย ดำ</span>（黑胡椒）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">โรย พริกไทย</span>（撒胡椒）
  - 例：โรย  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">พริกไทย</span>  ดำ  บน  สเต็ก（在牛排上撒黑胡椒）

- 44. **ขิง** [khǐng] 薑 (ginger)
  - 字構：諧音「坑」→ 薑讓你的嘴裡像有個坑（辛辣感）
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ขิง สด</span>（新鮮薑）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">นึ่ง กับ ขิง</span>（和薑一起蒸）
  - 例：นึ่ง  ปลา  กับ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ขิง</span>  สด  ให้  หอม（用新鮮薑蒸魚讓它香）

- 45. **ตะไคร้** [tà-khrai] 香茅 (lemongrass)
  - 字構：諧音「踏開來」→ 踏開來聞香茅的香氣
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ตะไคร้ หั่น</span>（切香茅）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ใส่ ตะไคร้ ใน ต้มยำ</span>（在酸辣湯中加香茅）
  - 例：ต้มยำ  ต้อง  ใส่  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ตะไคร้</span>  และ  ใบ  มะกรูด（酸辣湯一定要加香茅和檸檬葉）

<a id="味道"></a>
## 主題六：味道

- 46. **เปรี้ยว** [prîao] 酸 (sour)
  - 字構：諧音「比咬」→ 酸到比咬一樣讓你縮臉
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">รส เปรี้ยว</span>（酸味）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เปรี้ยว จี๊ด</span>（非常酸）
  - 例：มะนาว  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เปรี้ยว</span>  มาก  เลย（檸檬很酸耶）

- 47. **หวาน** [wǎan] 甜 (sweet)
  - 字構：諧音「萬」→ 萬種甜蜜的感覺
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">รส หวาน</span>（甜味）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">หวาน หอม</span>（香甜）
  - 例：ขนม  ไทย  ส่วนใหญ่  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">หวาน</span>  มาก（泰國甜點大多很甜）

- 48. **ขม** [khǒm] 苦 (bitter)
  - 字構：諧音「坑」→ 苦到像掉進坑裡
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">รส ขม</span>（苦味）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ขม นิดหน่อย</span>（微苦）
  - 例：กาแฟ  ดำ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ขม</span>  แต่  มี  กลิ่น  หอม（黑咖啡苦但香氣十足）

- 49. **เผ็ด** [phèt] 辣 (spicy/hot)
  - 字構：諧音「配」→ 辣椒配什麼都辣
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เผ็ด มาก</span>（很辣）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เผ็ด ร้อน</span>（辛辣）
  - 例：ต้มยำ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เผ็ด</span>  จนน้ำตาไหล（酸辣湯辣到流眼淚）

- 50. **เค็ม** [khem] 鹹 (salty)
  - 字構：諧音「咸」→ 台語「咸」就是鹹
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">รส เค็ม</span>（鹹味）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">เค็ม เกินไป</span>（太鹹）
  - 例：น้ำปลา  ทำให้  อาหาร  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">เค็ม</span>  พอดี（魚露讓食物鹹淡剛好）

- 51. **จืด** [jùeat] 淡/無味 (bland/tasteless)
  - 字構：諧音「絕」→ 絕對沒味道
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">รส จืด</span>（淡而無味）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">จืด ชืด</span>（非常無味）
  - 例：ซุป  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">จืด</span>  ไป  ต้อง  ใส่  เกลือ  เพิ่ม（湯太淡了要再加鹽）

- 52. **มัน** [man] 香濃/油潤 (rich/savory)
  - 字構：諧音「滿」→ 滿口香濃的油潤感
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">รส มัน</span>（香濃味）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">มัน อร่อย</span>（香濃好吃）
  - 例：แกง  กะทิ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">มัน</span>  และ  หอม  มาก（椰奶咖哩香濃又好吃）

- 53. **ฝาด** [fàat] 澀 (astringent)
  - 字構：諧音「法」→ 澀到像被法條卡住喉嚨
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">รส ฝาด</span>（澀味）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">ฝาด คอ</span>（卡喉澀感）
  - 例：กล้วย  ดิบ  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">ฝาด</span>  กิน  ไม่  ได้  เลย（未熟的香蕉很澀根本吃不了）

- 54. **อร่อย** [à-ràwy] 好吃 (delicious)
  - 字構：諧音「阿了喂」→ 阿了喂！好吃到驚呼
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">อร่อย มาก</span>（非常好吃）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กิน อร่อย</span>（吃得很開心）
  - 例：อาหาร  แม่  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">อร่อย</span>  ที่สุด  ใน  โลก（媽媽的菜是世界上最好吃的）

- 55. **กลมกล่อม** [klom-klàwm] 圓潤/醇厚 (well-rounded flavor)
  - 字構：กลม（圓）+ กล่อม（撫慰）= 圓潤撫慰人心的味道
  - 搭配：<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">รส กลมกล่อม</span>（醇厚圓潤）　<span style="background:#e8c547;color:#111;padding:1px 5px;border-radius:3px">กลมกล่อม ลงตัว</span>（味道恰到好處）
  - 例：แกง  นี้  รส  <span style="background:#4a90e2;color:#fff;padding:1px 5px;border-radius:3px">กลมกล่อม</span>  ดี  มาก（這道咖哩味道醇厚圓潤非常好）
