## 1. 泰文逐句翻譯

### 段落一：導言 · บทนำ

**泰文：** แผนที่  ที่  'ซ่อน'  ลำดับชั้น  
**中翻：** 隱藏階層的地圖

**泰文：** อ่านโลก  ผ่าน  Wallerstein — เมื่อ  ปัญหา  ไม่ใช่  ประเทศ  แต่  คือ  ระบบ  ที่  ประเทศ  อาศัยอยู่  
**中翻：** 透過 Wallerstein 讀懂世界——當問題不在於國家，而在於國家所處的系統。

重要單字：
- **แผนที่** [phǎen thîi] 地圖 (map) ← แผน (計畫) + ที่ (地方)
- **ซ่อน** [sôn] 隱藏 (hide) ← 單純詞
- **ลำดับชั้น** [lam dàp chán] 階層 (hierarchy) ← ลำดับ (順序) + ชั้น (層)
- **ปัญหา** [pan hǎa] 問題 (problem) ← 諧音「盤哈」→ 盤問有問題
- **อาศัยอยู่** [aa sǎi yùu] 居住、依存 (reside in) ← อาศัย (依靠) + อยู่ (在)

---

**泰文：** ลอง  นึกภาพ  โลก  ไม่ใช่  ใน  ฐานะ  ประเทศ  หลาย  ร้อย  ประเทศ  ที่  แข่งขัน  กัน  อย่าง  เท่าเทียม  แต่  เป็น  ระบบ  เดียว  ที่  มี  ลำดับชั้น  ซ่อน  อยู่  ภายใน  
**中翻：** 試著想像世界，不是以數百個平等相互競爭的國家來看待，而是一個內部隱藏著階層的單一系統。

重要單字：
- **นึกภาพ** [núek phâap] 想像 (imagine) ← นึก (想) + ภาพ (畫面)
- **โลก** [lôok] 世界 (world) ← 諧音英語 look → 看出去就是世界
- **ฐานะ** [thǎa ná] 地位、身份 (status) ← 諧音「他那」→「他那個」地位
- **ประเทศ** [prà thêet] 國家 (country) ← ประ + เทศ (梵語 deśa)
- **แข่งขัน** [khàeng khǎn] 競爭 (compete) ← แข่ง (賽) + ขัน (競)
- **เท่าเทียม** [thâo thiam] 平等 (equal) ← เท่า (相等) + เทียม (並列)
- **ระบบ** [rá bòp] 系統 (system) ← 諧音「啊爆」→ 系統爆了
- **ภายใน** [phaai nai] 內部 (inside) ← ภาย (方面) + ใน (裡)

---

**泰文：** บาง  ประเทศ  อยู่  บนสุด  บาง  ประเทศ  อยู่  ตรงกลาง  และ  บาง  ประเทศ  อยู่  ล่างสุด  
**中翻：** 有些國家在最頂端，有些國家在中間，有些國家在最底層。

重要單字：
- **บาง** [baang] 某些 (some) ← 單純詞
- **บนสุด** [bon sùt] 最上面 (topmost) ← บน (上) + สุด (最)
- **ตรงกลาง** [trong klaang] 中間 (in the middle) ← ตรง (正) + กลาง (中)
- **ล่างสุด** [lâang sùt] 最下面 (bottommost) ← ล่าง (下) + สุด (最)

---

**泰文：** ลำดับชั้น  นั้น  ไม่  ได้  เกิด  จาก  ความขยัน  หรือ  ความฉลาด  ของ  ประชาชน  ใน  แต่ละ  ประเทศ  แต่  เกิด  จาก  โครงสร้าง  ของ  ระบบ  ที่  กำหนด  ว่า  ใคร  ได้  ประโยชน์  จาก  การค้า  โลก  และ  ใคร  เสียเปรียบ  
**中翻：** 這個階層不是由各國人民的勤勞或聰明所造成的，而是由決定誰從全球貿易中獲益、誰處於劣勢的系統結構所造成的。

重要單字：
- **เกิดจาก** [kòet jàak] 源自 (arise from) ← เกิด (產生) + จาก (從)
- **ความขยัน** [khwaam khà yǎn] 勤勞 (diligence) ← ความ + ขยัน
- **ความฉลาด** [khwaam chà làat] 聰明 (intelligence) ← ความ + ฉลาด
- **ประชาชน** [prà chaa chon] 人民 (people) ← ประชา (民眾) + ชน (人)
- **โครงสร้าง** [khroong sâang] 結構 (structure) ← โครง (框架) + สร้าง (建造)
- **กำหนด** [kam nòt] 決定、規定 (determine) ← 諧音「卡腦」→ 腦袋卡住=被決定
- **ประโยชน์** [prà yòot] 利益 (benefit) ← 諧音「跑優」→ 跑上去才有利益
- **การค้า** [kaan kháa] 貿易 (trade) ← การ + ค้า (做生意)
- **เสียเปรียบ** [sǐa prìap] 處於劣勢 (disadvantage) ← เสีย (失去) + เปรียบ (比較)

---

**泰文：** นี่  คือ  จุดเริ่มต้น  ของ  Immanuel Wallerstein  ใน  The Modern World-System  ที่  ตีพิมพ์  ใน  ปี  2517  และ  ยังคง  เป็น  หนึ่ง  ใน  งาน  ที่  อ่าน  โลก  ได้  คม  ที่สุด  เท่าที่  เคย  มี  มา  
**中翻：** 這就是 Wallerstein 在 1974 年出版的《現代世界體系》的起點，至今仍是解讀世界最深刻的著作之一。

重要單字：
- **จุดเริ่มต้น** [jùt rôem tôn] 起點 (starting point) ← จุด (點) + เริ่มต้น (開始)
- **ตีพิมพ์** [tii phim] 出版 (publish) ← ตี (打) + พิมพ์ (印刷)
- **คม** [khom] 銳利、深刻 (sharp) ← 單純詞

### 段落二：Core, Periphery 與 Semiperiphery · Core, Periphery และ Semiperiphery

**泰文：** วอลเลอร์สไตน์  แบ่ง  <mark>ประเทศ</mark>  ใน  <mark>โลก</mark>  ออก  เป็น  สาม  กลุ่ม  ไม่ใช่  ตาม  ระดับ  การพัฒนา  แต่  ตาม  หน้าที่  ใน  <mark>ระบบ</mark>  <mark>โลก</mark>  
**中翻：** Wallerstein 將世界上的國家分為三組——不是按發展程度，而是按在世界體系中的功能。

重要單字：
- **แบ่ง** [bàeng] 分開 (divide) ← 單純詞
- **กลุ่ม** [klùm] 組、群 (group) ← 單純詞
- **ระดับ** [rá dàp] 程度、等級 (level) ← 單純詞
- **การพัฒนา** [kaan phát thá naa] 發展 (development) ← การ + พัฒนา (巴利語)
- **หน้าที่** [nâa thîi] 職責、功能 (duty, function) ← หน้า (面) + ที่ (的)

---

**泰文：** กลุ่ม  แรก  คือ  Core  หรือ  ศูนย์กลาง  ซึ่ง  ไม่  ได้  หมายความ  แค่  ว่า  รวย  กว่า  แต่  หมายความ  ว่า  เป็น  ผู้  <mark>กำหนด</mark>  กฎเกณฑ์  ของ  <mark>ระบบ</mark>  ครอง  เทคโนโลยี  การผลิต  ขั้นสูง  ควบคุม  <mark>ระบบ</mark>  การเงิน  และ  ผลิต  สินค้า  ที่  มี  มูลค่าเพิ่ม  สูง  
**中翻：** 第一組是 Core（核心），不僅意味著更富有，而是制定系統規則的人、掌控高端生產技術、控制金融體系，並生產高附加值商品。

重要單字：
- **ศูนย์กลาง** [sǔun klaang] 中心 (center) ← ศูนย์ (梵語) + กลาง (中間)
- **หมายความ** [mǎai khwaam] 意味著 (mean) ← หมาย (指) + ความ (意義)
- **กฎเกณฑ์** [kòt keen] 規則 (rules) ← กฎ (規則) + เกณฑ์ (標準)
- **ครอง** [khraawng] 掌控 (dominate) ← 單純詞
- **การผลิต** [kaan phà lìt] 生產 (production) ← การ + ผลิต
- **ขั้นสูง** [khân sǔung] 高級的 (advanced) ← ขั้น (階段) + สูง (高)
- **ควบคุม** [khûap khum] 控制 (control) ← ควบ (合) + คุม (管)
- **การเงิน** [kaan ngoen] 金融 (finance) ← การ + เงิน (錢)
- **มูลค่าเพิ่ม** [muun khâa phôem] 附加值 (added value) ← มูลค่า + เพิ่ม (增加)

---

**泰文：** <mark>ประเทศ</mark>  กลุ่ม  นี้  ดึง  มูลค่า  ออก  จาก  ส่วน  อื่น  ของ  <mark>ระบบ</mark>  เข้ามา  หา  ตัวเอง  อย่าง  สม่ำเสมอ  
**中翻：** 這類國家持續從系統的其他部分汲取價值，源源不斷地流向自身。

重要單字：
- **ดึง** [dueng] 抽取、吸引 (extract, pull) ← 單純詞
- **ส่วน** [sùan] 部分 (part) ← 單純詞
- **เข้ามา** [khâo maa] 進入、流向 (come in) ← เข้า (進) + มา (來)
- **สม่ำเสมอ** [sà mâm sà mǒe] 持續、一致 (consistently) ← 單純詞

---

**泰文：** กลุ่ม  ที่สอง  คือ  Periphery  หรือ  รอบนอก  ซึ่ง  ทำ  หน้าที่  ส่งออก  วัตถุดิบ  แรงงาน  ราคาถูก  และ  สินค้า  ที่  มี  มูลค่าเพิ่ม  ต่ำ  
**中翻：** 第二組是 Periphery（邊陲），負責出口原材料、廉價勞動力和低附加值商品。

重要單字：
- **รอบนอก** [rôop nôok] 外圍 (periphery) ← รอบ (周圍) + นอก (外)
- **ส่งออก** [sòng òok] 出口 (export) ← ส่ง (送) + ออก (出)
- **วัตถุดิบ** [wát thù dìp] 原材料 (raw material) ← วัตถุ (巴利語) + ดิบ (生)
- **แรงงาน** [raeng ngaan] 勞動力 (labor) ← แรง (力) + งาน (工作)
- **ราคาถูก** [raa khaa thùuk] 便宜的 (cheap) ← ราคา (價格) + ถูก (便宜)
- **สินค้า** [sǐn kháa] 商品 (goods) ← สิน (財) + ค้า (貿易)

---

**泰文：** <mark>ประเทศ</mark>  กลุ่ม  นี้  ไม่  ยากจน  เพราะ  ขาด  ทรัพยากร  หรือ  ขาด  <mark>ความขยัน</mark>  แต่  ยากจน  เพราะ  <mark>โครงสร้าง</mark>  ของ  การแลกเปลี่ยน  ใน  <mark>ระบบ</mark>  <mark>โลก</mark>  ทำให้  มูลค่า  ที่  พวกเขา  สร้าง  ไหล  ออก  ไป  สู่  Core  ตลอดเวลา  
**中翻：** 這些國家並不是因為缺乏資源或不夠勤勞而貧窮，而是因為世界體系中的交換結構使他們創造的價值持續流向核心。

重要單字：
- **ยากจน** [yâak jon] 貧窮 (poor) ← ยาก (困難) + จน (窮)
- **ขาด** [khàat] 缺乏 (lack) ← 單純詞
- **ทรัพยากร** [sáp phá yaa kawn] 資源 (resource) ← 諧音「薩補」→ 補充資源
- **แลกเปลี่ยน** [lâek plìan] 交換 (exchange) ← แลก (換) + เปลี่ยน (變)
- **มูลค่า** [muun khâa] 價值 (value) ← มูล (巴利語) + ค่า (值)
- **ไหล** [lǎi] 流動 (flow) ← 單純詞
- **ตลอดเวลา** [tà lòot wee laa] 持續 (all the time) ← ตลอด (貫穿) + เวลา (時間)

---

**泰文：** กลุ่ม  ที่สาม  คือ  Semiperiphery  หรือ  กึ่ง  รอบนอก  และ  นี่  คือ  ตำแหน่ง  ที่  น่าสนใจ  ที่สุด  ใน  <mark>ระบบ</mark>  
**中翻：** 第三組是 Semiperiphery（半邊陲），這是系統中最有趣的位置。

重要單字：
- **กึ่ง** [kùeng] 半 (semi-) ← 單純詞
- **ตำแหน่ง** [tam nàeng] 位置 (position) ← 諧音「探能」→ 探測位置
- **น่าสนใจ** [nâa sǒn jai] 有趣的 (interesting) ← น่า (值得) + สนใจ (感興趣)

---

**泰文：** <mark>ประเทศ</mark>  กลุ่ม  นี้  อยู่  <mark>ตรงกลาง</mark>  พวกเขา  มี  อุตสาหกรรม  มี  ชนชั้นกลาง  มี  รัฐ  ที่  เข้มแข็ง  พอควร  แต่  ยัง  ไม่  ถึง  ระดับ  ที่  จะ  <mark>กำหนด</mark>  กฎเกณฑ์  ของ  <mark>ระบบ</mark>  ได้  
**中翻：** 這些國家處於中間位置，他們有工業、有中產階級、有相當強大的國家機器，但還不到能制定系統規則的程度。

重要單字：
- **อุตสาหกรรม** [ùt sǎa hà kam] 工業 (industry) ← sāha=熱情+kamma=行業 → 有熱情的行業
- **ชนชั้นกลาง** [chon chán klaang] 中產階級 (middle class) ← ชน + ชั้น + กลาง
- **รัฐ** [rát] 國家、政府 (state) ← 諧音「辣特」→ 辣特色=國家特色
- **เข้มแข็ง** [khêm khǎeng] 強大 (strong) ← เข้ม (濃) + แข็ง (硬)
- **พอควร** [phaw khuan] 相當地 (moderately) ← พอ (夠) + ควร (應該)

---

**泰文：** หน้าที่  สำคัญ  ของ  Semiperiphery  ใน  <mark>ระบบ</mark>  คือ  การ  ทำ  หน้าที่  เป็น  "ตัวกลาง"  ดูดซับ  แรงกดดัน  ทาง  การเมือง  จาก  Periphery  ไม่  ให้  มัน  พุ่ง  ตรง  ไป  ยัง  Core  โดยตรง  
**中翻：** Semiperiphery 在系統中的重要功能是充當「中間人」，吸收來自邊陲的政治壓力，使其不會直接衝向核心。

重要單字：
- **สำคัญ** [sǎm khan] 重要 (important) ← 諧音「三砍」→ 砍三刀=很重要
- **ตัวกลาง** [tua klaang] 中間人 (intermediary) ← ตัว (個體) + กลาง (中)
- **ดูดซับ** [dùut sáp] 吸收 (absorb) ← ดูด (吸) + ซับ (吸附)
- **แรงกดดัน** [raeng kòt dan] 壓力 (pressure) ← แรง (力) + กดดัน (施壓)
- **การเมือง** [kaan mueang] 政治 (politics) ← การ + เมือง
- **พุ่ง** [phûng] 衝 (rush) ← 單純詞

### 段落三：為什麼系統位置比政府政策重要 · ทำไมตำแหน่งในระบบถึงสำคัญกว่านโยบายรัฐบาล

**泰文：** สิ่ง  ที่  ทำให้  วอลเลอร์สไตน์  แตกต่าง  จาก  นักวิเคราะห์  นโยบาย  ทั่วไป  คือ  เขา  เชื่อ  ว่า  <mark>ตำแหน่ง</mark>  ของ  <mark>ประเทศ</mark>  ใน  <mark>ระบบ</mark>  <mark>โลก</mark>  มี  ผล  ต่อ  ชะตากรรม  ของ  <mark>ประเทศ</mark>  นั้น  มากกว่า  นโยบาย  <mark>ภายใน</mark>  เสีย  อีก  
**中翻：** Wallerstein 與一般政策分析師不同在於，他認為國家在世界體系中的位置對命運的影響超過國內政策。

重要單字：

**泰文：** สิ่ง  ที่  น่าสังเกต  คือ  Core  ไม่  ได้  ลงโทษ  ทุก  <mark>ประเทศ</mark>  ที่  ละเมิด  บรรทัดฐาน  ระหว่าง  <mark>ประเทศ</mark>  มัน  ลงโทษ  เฉพาะ  <mark>ประเทศ</mark>  ที่  <mark>ท้าทาย</mark>  <mark>ลำดับชั้น</mark>  ของ  <mark>ระบบ</mark>  
**中翻：** 值得注意的是，核心國家並不懲罰所有違反國際規範的國家，只懲罰那些挑戰系統階層的國家。

重要單字：
- **น่าสังเกต** [nâa sǎng kèet] 值得注意的 (noteworthy) ← น่า + สังเกต (觀察)
- **ลงโทษ** [long thôot] 懲罰 (punish) ← ลง (施加) + โทษ (刑罰)
- **บรรทัดฐาน** [ban thàt thǎan] 規範 (norm) ← 諧音「班打坦」→ 打出坦率規範
- **เฉพาะ** [chà phaw] 僅、特定 (only, specific) ← 單純詞

---

**泰文：** <mark>ประเทศ</mark>  ที่  ทำ  สิ่ง  เดียวกัน  แต่  ยังคง  อยู่  ใน  <mark>ตำแหน่ง</mark>  ที่  <mark>ระบบ</mark>  <mark>กำหนด</mark>  ให้  มัก  จะ  ได้รับ  การยกเว้น  ความ  <mark>ไม่สม่ำเสมอ</mark>  นั้น  เอง  คือ  หลักฐาน  ที่  บอก  ว่า  เรา  กำลัง  เผชิญ  กับ  ตรรกะ  ของ  การ  รักษา  <mark>ระบบ</mark>  ไม่ใช่  ตรรกะ  ของ  หลักการ  
**中翻：** 做同樣事情但仍留在系統指定位置的國家往往被豁免。這種不一致本身就是證據，說明我們面對的是維護系統的邏輯，而非原則的邏輯。

重要單字：
- **การยกเว้น** [kaan yók wên] 豁免 (exemption) ← การ + ยกเว้น (免除)
- **ไม่สม่ำเสมอ** [mâi sà mâm sà mǒe] 不一致 (inconsistent) ← ไม่ + สม่ำเสมอ
- **หลักฐาน** [làk thǎan] 證據 (evidence) ← หลัก (柱/基礎) + ฐาน (基礎)
- **เผชิญ** [phà choen] 面對、面臨 (face, confront) ← 諧音「怕千」→ 怕了還是要面對
- **ตรรกะ** [tan rá ká] 邏輯 (logic) ← 梵語辯論詞 tarka → 辯論講邏輯
- **หลักการ** [làk kaan] 原則 (principle) ← หลัก (柱) + การ (事)

### 段落六：制裁作為維持階層的機制 · การคว่ำบาตรในฐานะกลไกรักษาลำดับชั้น

**泰文：** การคว่ำบาตร  <mark>ทาง</mark>  เศรษฐกิจ  มัก  ถูก  <mark>นำเสนอ</mark>  ว่า  เป็น  เครื่องมือ  <mark>ทาง</mark>  การทูต  ที่  ใช้  บังคับ  ให้  <mark>ประเทศ</mark>  ที่  ละเมิด  กฎหมาย  ระหว่าง  <mark>ประเทศ</mark>  ปรับ  พฤติกรรม  
**中翻：** 經濟制裁通常被呈現為一種外交工具，用來迫使違反國際法的國家調整行為。

重要單字：
- **คว่ำบาตร** [khwâm bàat] 制裁 (sanction) ← คว่ำ (翻覆) + บาตร (缽)
- **เศรษฐกิจ** [sèet thà kìt] 經濟 (economy) ← 諧音「賽特」→ 競賽特區=經濟
- **เครื่องมือ** [khrûeang muue] 工具 (tool) ← เครื่อง (器具) + มือ (手)
- **การทูต** [kaan thûut] 外交 (diplomacy) ← การ + ทูต (使者)
- **บังคับ** [bang kháp] 強制 (force) ← 諧音「邦卡」→ 被卡死=強制
- **ละเมิด** [lá mòet] 違反 (violate) ← 諧音「拉滅」→ 拉掉規則=違反
- **กฎหมาย** [kòt mǎai] 法律 (law) ← กฎ (規則) + หมาย (記號)
- **พฤติกรรม** [phrúe tì kam] 行為 (behavior) ← kam=巴利語「業」→ 造業=行為

---

**泰文：** คำอธิบาย  นี้  ฟัง  ดู  สมเหตุสมผล  และ  เป็นกลาง  แต่  เมื่อ  เรา  ดู  ว่า  ใคร  ถูก  คว่ำบาตร  และ  ใคร  ไม่  ถูก  คว่ำบาตร  <mark>รูปแบบ</mark>  ที่  ปรากฏ  ขึ้น  บอก  เรื่อง  ที่  ต่าง  ออก  ไป  โดยสิ้นเชิง  
**中翻：** 這個解釋聽起來合理且中立，但當我們看看誰被制裁、誰沒被制裁，浮現的模式卻說著完全不同的故事。

重要單字：
- **คำอธิบาย** [kham à thí baai] 說明、解釋 (explanation) ← คำ (話) + อธิบาย (說明)
- **สมเหตุสมผล** [sǒm hèet sǒm phǒn] 合理的 (reasonable) ← สม (符合) + เหตุ (原因) + ผล (結果)
- **เป็นกลาง** [pen klaang] 中立的 (neutral) ← เป็น (是) + กลาง (中間)
- **รูปแบบ** [rûup bàep] 模式 (pattern) ← รูป (形) + แบบ (樣)
- **ปรากฏ** [praa kòt] 浮現、出現 (appear) ← 諧音「爬鍋」→ 從水中爬出=浮現
- **โดยสิ้นเชิง** [dooi sîn choeng] 完全地 (entirely) ← โดย (由) + สิ้นเชิง (徹底)

---

**泰文：** <mark>ประเทศ</mark>  ที่  ถูก  คว่ำบาตร  หนัก  ที่สุด  ใน  <mark>โลก</mark>  คือ  <mark>ประเทศ</mark>  ที่  พยายาม  สร้าง  <mark>ระบบ</mark>  เศรษฐกิจ  และ  <mark>การเมือง</mark>  ที่  เป็น  อิสระ  จาก  <mark>ระเบียบ</mark>  ที่  Core  <mark>กำหนด</mark>  ไม่ใช่  <mark>ประเทศ</mark>  ที่  ละเมิด  สิทธิมนุษยชน  มาก  ที่สุด  หรือ  พัฒนา  อาวุธ  <mark>อันตราย</mark>  ที่สุด  
**中翻：** 世界上受制裁最嚴厲的國家，是那些試圖建立獨立於核心秩序之外的經濟和政治體系的國家——而不是那些違反人權最多或發展最危險武器的國家。

重要單字：
- **อิสระ** [ìt sà rà] 自由、獨立 (free, independent) ← 諧音「以色列」→ 自由之地
- **สิทธิมนุษยชน** [sìt thì má nút yá chon] 人權 (human rights) ← สิทธิ (權利) + มนุษย (人類) + ชน (人)
- **อาวุธ** [aa wút] 武器 (weapon) ← 諧音「啊物」→ 啊這個武器
- **อันตราย** [an tà raai] 危險 (dangerous) ← 諧音「暗的來」→ 暗中危險逼近

---

**泰文：** ถ้า  <mark>หลักการ</mark>  เป็น  เรื่อง  สิทธิมนุษยชน  หรือ  อาวุธ  อย่าง  ที่  <mark>ประกาศ</mark>  เรา  จะ  ต้อง  อธิบาย  ได้  ว่า  ทำไม  <mark>พันธมิตร</mark>  ของ  Core  ที่  ทำ  สิ่ง  เดียวกัน  ถึง  ไม่  ถูก  <mark>ลงโทษ</mark>  ใน  <mark>ระดับ</mark>  เดียวกัน  
**中翻：** 如果原則真的是關於人權或武器，我們就必須能解釋，為什麼做了同樣事情的核心盟友沒有受到同等的懲罰。

重要單字：
- **อธิบาย** [à thí baai] 解釋 (explain) ← 諧音「啊替白」→ 替人說清白=解釋
- **พันธมิตร** [phan thá mít] 盟友 (ally) ← mitra=梵語「友」→ 友人之盟
- **ระดับเดียวกัน** [rá dàp diaw kan] 同等程度 (same level) ← ระดับ + เดียวกัน (相同)

---

**泰文：** วอลเลอร์สไตน์  ตอบ  ว่า  เพราะ  การคว่ำบาตร  ไม่ใช่  กลไก  <mark>ทาง</mark>  กฎหมาย  แต่  คือ  กลไก  <mark>ทาง</mark>  <mark>ระบบ</mark>  <mark>หน้าที่</mark>  ของ  มัน  คือ  ส่ง  สัญญาณ  ว่า  การ  ออก  นอก  <mark>ลำดับชั้น</mark>  ของ  <mark>ระบบ</mark>  มี  ราคา  และ  ราคา  นั้น  ถูก  <mark>ออกแบบ</mark>  ให้  สูง  พอ  ที่  จะ  ทำให้  <mark>ประเทศ</mark>  อื่น ๆ  ใน  Semiperiphery  คิดหนัก  ก่อน  จะ  ทำ  เช่นเดียวกัน  
**中翻：** Wallerstein 的回答是：制裁不是法律機制，而是系統機制。它的功能是發出信號：脫離系統階層是要付出代價的，而且代價被刻意設計得足夠高，讓其他半邊陲國家在採取同樣行動之前三思。

重要單字：
- **กลไก** [konkai] 機制 (mechanism) ← กล (機械) + ไก (齒輪)
- **สัญญาณ** [sǎn yaan] 信號 (signal) ← 諧音「三言」→ 三言兩語=訊號
- **ราคา** [raa khaa] 代價 (price, cost) ← 諧音「啦卡」→ 被卡住要付代價
- **คิดหนัก** [khít nàk] 三思 (think hard) ← คิด (想) + หนัก (重)
- **เช่นเดียวกัน** [chên diaw kan] 同樣地 (likewise) ← เช่น (如) + เดียวกัน (相同)

---

### 段落七：過渡期與重新結盟 · ช่วงเปลี่ยนผ่านและการจัดแนวใหม่

**泰文：** สิ่ง  ที่  น่าจับตา  ใน  ช่วง  <mark>เปลี่ยนผ่าน</mark>  <mark>อำนาจ</mark>  นำ  คือ  <mark>พฤติกรรม</mark>  ของ  <mark>ประเทศ</mark>  Semiperiphery  <mark>ในวัฏจักร</mark>  ของ  <mark>อำนาจ</mark>  นำ  <mark>ประเทศ</mark>  เหล่านี้  มัก  พยายาม  "เดิมพัน"  กับ  <mark>มหาอำนาจ</mark>  ที่  กำลัง  ขึ้น  มา  
**中翻：** 在霸權過渡期值得關注的，是半邊陲國家的行為——它們通常試圖與正在崛起的大國「押注」。

重要單字：
- **น่าจับตา** [nâa jàp taa] 值得關注 (worth watching) ← น่า + จับตา (盯住看)
- **พฤติกรรม** [phrúe tì kam] 行為 (behavior) ← kam=巴利語「業」→ 造業=行為
- **เดิมพัน** [doem phan] 押注、賭注 (bet, wager) ← เดิม (原始) + พัน (綁)

---

**泰文：** <mark>ขณะเดียวกัน</mark>  ก็  รักษา  ความสัมพันธ์  กับ  <mark>มหาอำนาจ</mark>  เดิม  เอา  ไว้  บ้าง  เพราะ  ความผิดพลาด  ใน  การ  เลือกข้าง  ใน  ช่วง  <mark>เปลี่ยนผ่าน</mark>  มี  <mark>ผล</mark>  <mark>ยาวนาน</mark>  หลาย  <mark>ทศวรรษ</mark>  
**中翻：** 同時也保留一些與舊霸權的關係，因為在過渡期選邊站的錯誤，影響可能延續數十年。

重要單字：
- **ความสัมพันธ์** [khwaam sǎm phan] 關係 (relationship) ← ความ + สัมพันธ์ (來自巴利語)
- **รักษา** [rák sǎa] 維護、保持 (maintain) ← rák=泰語「愛」→ 愛護即維護
- **ความผิดพลาด** [khwaam phìt phlâat] 錯誤 (mistake) ← ความ + ผิดพลาด
- **เลือกข้าง** [lûeak khâang] 選邊站 (choose sides) ← เลือก (選) + ข้าง (邊)

---

**泰文：** วอลเลอร์สไตน์  จะ  อ่าน  การ  ที่  <mark>ประเทศ</mark>  ใน  Global South  หลาย  <mark>ประเทศ</mark>  ปฏิเสธ  ที่  จะ  เข้าข้าง  <mark>สหรัฐ</mark>  ใน  <mark>ความขัดแย้ง</mark>  กับ  รัสเซีย  และ  จีน  ไม่ใช่  ว่า  เพราะ  พวกเขา  "ไม่  มี  <mark>หลักการ"</mark>  อย่าง  ที่  สื่อ  ตะวันตก  มัก  กล่าวหา  
**中翻：** Wallerstein 會這樣解讀：許多全球南方國家拒絕在美俄中衝突中選邊站，不是因為他們「沒有原則」——這是西方媒體常見的指責。

重要單字：
- **ปฏิเสธ** [pà tì sèet] 拒絕 (refuse) ← 諧音「怕踢死」→ 踢走=拒絕
- **เข้าข้าง** [khâo khâang] 選邊站 (take sides) ← เข้า (進入) + ข้าง (邊)
- **สื่อ** [sùue] 媒體 (media) ← 單純詞
- **ตะวันตก** [tà wan tòk] 西方 (west) ← ตะวัน (太陽) + ตก (落)
- **กล่าวหา** [klàao hǎa] 指責、指控 (accuse) ← กล่าว (說) + หา (找)

---

**泰文：** แต่  เพราะ  พวกเขา  กำลัง  อ่าน  <mark>วัฏจักร</mark>  <mark>อำนาจ</mark>  นำ  และ  พยายาม  รักษา  ทางเลือก  เอา  ไว้  ให้  มาก  ที่สุด  ใน  ช่วง  ที่  ไม่มี  ใคร  รู้  ว่า  <mark>ระเบียบ</mark>  <mark>โลก</mark>  ใหม่  จะ  หน้าตา  เป็น  อย่างไร  
**中翻：** 而是因為他們正在解讀霸權循環，並在沒有人知道新世界秩序將是什麼面貌的時候，盡可能保留自己的選擇空間。

重要單字：
- **ทางเลือก** [thaang lûeak] 選擇 (option) ← ทาง (路) + เลือก (選)
- **หน้าตา** [nâa taa] 面貌、樣子 (appearance, face) ← หน้า (臉) + ตา (眼)
- **พยายาม** [phá yaa yaam] 努力 (try) ← 諧音「帕藥樣」→ 像吃藥一樣努力

### 結語 · บทสรุป

**泰文：** สิ่ง  ที่  วอลเลอร์สไตน์  ให้  เรา  และ  ไม่  อาจ  หา  จาก  ที่ไหน  ได้  คือ  <mark>แผนที่</mark>  <mark>แผนที่</mark>  ที่  บอก  ว่า  <mark>ประเทศ</mark>  ต่าง ๆ  ไม่  ได้  อยู่  ใน  <mark>โลก</mark>  แบนราบ  ที่  <mark>เท่าเทียม</mark>  กัน  แต่  อยู่  ใน  <mark>ระบบ</mark>  ที่  มี  <mark>ลำดับชั้น</mark>  
**中翻：** Wallerstein 給予我們的、無法從其他地方找到的，是一幅地圖——告訴我們各國並非處於平坦平等的世界，而是處於一個有階層的系統中。

**泰文：** และ  การ  เข้าใจ  <mark>ลำดับชั้น</mark>  นั้น  คือ  <mark>เงื่อนไข</mark>  เบื้องต้น  ใน  การ  เข้าใจ  ว่า  <mark>ความขัดแย้ง</mark>  ใด ๆ  กำลัง  เกิดขึ้น  เพื่อ  อะไร  และ  รับใช้  ใคร  
**中翻：** 而理解那個階層，是理解任何衝突為何發生、為誰服務的基本前提。

重要單字：
- **แบนราบ** [baen râap] 平坦的 (flat) ← แบน (扁) + ราบ (平)
- **เงื่อนไขเบื้องต้น** [ngûean khǎi bûeang tôn] 基本前提 (prerequisite) ← เงื่อนไข (條件) + เบื้องต้น (初步)
- **รับใช้** [ráp chái] 服務、效勞 (serve) ← รับ (接受) + ใช้ (使用)

## 3. 近似字整理

- **ซ่อน** [sôn] 隱藏 vs **ซ้อน** [són] 重疊
  - 聲調不同：ซ่อน 低聲調=隱藏；ซ้อน 高聲調=重疊

- **แบ่ง** [bàeng] 分開 vs **แบน** [baen] 扁的
  - 尾音不同：-ง vs -น

- **ครอง** [khraawng] 統治 vs **คลอง** [khlaawng] 運河
  - ร vs ล 的區別

- **ผล** [phǒn] 結果 vs **พล** [phon] 力量
  - ผ vs พ：都是送氣音但書寫不同

- **อำนาจ** [am nâat] 權力 vs **อาณาจักร** [aa naa jàk] 王國
  - 同源：อำนาจ=抽象權力；อาณาจักร=統治領域

- **กฎ** [kòt] 規則 vs **กด** [kòt] 按、壓
  - 同音不同字：กฎ 有 ฎ；กด 有 ด

- **ราคา** [raa khaa] 價格 vs **รักษา** [rák sǎa] 治療/維護
  - 前兩音節相似但意思完全不同

- **เสีย** [sǐa] 失去/壞掉 vs **เสี่ย** [sìa] 擲骰子
  - 聲調不同

- **นำ** [nam] 領導 vs **น้ำ** [náam] 水
  - 聲調+長短音不同：นำ 平聲短音；น้ำ 高聲調

- **ดัน** [dan] 推動 vs **ดัน** [dan] 頂嘴（口語）
  - 同音同字不同意：語境決定含義

- **ยอม** [yaawm] 順從 vs **ยืม** [yuem] 借
  - 母音不同：-o- vs -ue-

- **เดิมพัน** [doem phan] 押注 vs **ผูกพัน** [phùuk phan] 牽掛/束縛
  - 前綴不同，พัน 相同但整體意思不同
